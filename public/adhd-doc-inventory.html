So What

84件は多く見えるが、Tier 1の47件のうち16件はDAY講義の教材セット（8DAY×スライド+ノート）。実質的に「ゼロから設計が必要」な文書は約25件。残りはテンプレートの適用やツール開発（スクリーニングWebフォーム等）で対応できる。

Section 01
準備ロードマップ — プレローンチ12週の配分

プレローンチ12週をPhase 1-3に分割し、Tier 1の47件を段階的に準備。 Coach（緑）とEngineer（シアン）の分担を明確化。

Phase 1 — 基盤構築（W1-4）
Coach: A1-A4 法務文書ドラフト / B4 インテーク面談シート / E1-E5 セッション台本ドラフト（4本柱フレームワーク）
Engineer: B1-B3 スクリーニングWebフォーム構築 / F1-F2 運営チェックリスト / A5 特商法表記
共同: A6 医療機関リスト作成 / B5-B6 除外基準・通知テンプレート
Phase 2 — コンテンツ開発（W5-8）
Coach: D1-D16 DAY講義スライド＋ファシリテーターノート（8DAY分）
Engineer: C1-C3 オンボーディングガイド / F4-F5 トラッキング＆アラート設計
共同: C4 バディシステム説明書 / C6 緊急連絡先カード
Phase 3 — 受入準備（W9-12）
Coach: 講義リハーサル / セッション台本最終化（4本柱） / A3 個人情報保護方針最終化
Engineer: C5 プレアセスメントフォーム構築 / ツール統合テスト
共同: F3 コンティンジェンシープラン / F6-F8 ファウンダー運営ルール
Phase 4 — 走りながら整備（Cohort 1期間中）
G グループセッション資料 / H セルフマニュアルテンプレート / I 効果測定尺度 / J マーケティング素材
※ Cohort 1の実体験を反映しながら順次整備
Phase 5 — 振返りと制度化（Cohort 1完了後）
K 卒業・アフターフォロー / L イテレーション文書 / M 創業者間合意の見直し
※ Cohort 1の検証結果に基づき策定
Key Insight

Coach = コンテンツ（講義・セッション台本）、Engineer = ツール＆仕組み（Webフォーム・ダッシュボード・ガイド類）で明確に分担。法務・安全管理は共同作業。この分担がプレローンチ12週の負荷を均等化する。
