### 2.1 X（Twitter）アルゴリズム — 2025-2026

#### コアアーキテクチャ
- **モデル**: Grok搭載トランスフォーマー（2023年オープンソース → **2026年1月20日に完全オープンソース化**）
- **更新頻度**: **4週間サイクル**で定期更新（開発者ノート付き）
- **候補生成**: ~1,500ツイートを選出 → 重み付きスコアリング → For Youに表示
- **ソース配分**: In-Network 50% / Out-of-Network 50%
- **トーン分析**: Grokが投稿のトーンを分析。**ポジティブ/建設的コンテンツ = 配信拡大、ネガティブ/攻撃的 = 抑制**

#### ランキングシグナル（重み順）

| シグナル | 重み | 実践的意味 |
|---------|------|----------|
| **リプライへの返信（会話）** | 最高（1.0基準で他の数倍） | 投稿にリプライ誘導質問を必ず入れる |
| **エンゲージメント率** | いいね 0.5x / RT 1.0x / リプライ 27x / ブクマ 4x | RT < リプライ。議論を生む投稿が最強 |
| **ユーザー信頼度（TweepCred）** | PageRank類似。フォロー/フォロワー比、アカウント年齢 | 相互フォロー整理、権威あるアカウントとの会話 |
| **コンテンツ鮮度** | 半減期6時間 | 投稿後6時間以内のエンゲージメントが勝負 |
| **メディア付き** | 画像2x / 動画6x（テキストonly比） | 画像 or 動画を必ず添付 |

#### Premium（有料プラン）の効果
- **配信ブースト**: Premium = **10倍**、Premium+ = **25倍**のインプレッション（Buffer 1800万投稿分析）
- **返信ブースト**: Premium+ユーザーの返信がスレッド最上部に表示
- **リンクペナルティ回避**: 2026年3月以降、非Premiumの外部リンク投稿は**エンゲージメントほぼゼロ**
- **長文投稿**: 25,000文字まで（スレッド代替として活用可能）
- **ROI**: $8/月で10倍リーチ。ビジネス用途なら必須

#### 2026年2月 重要アップデート: Communities公開化
- **全Community投稿が公開表示に**（メンバー外のFor Youフィードにも表示）
- 0-1Kフォロワー段階では**投稿の100%をCommunities経由**がベスト戦術
- Build in Public Community（18万人+）のような大規模Communityに参加→初期リーチ獲得

#### Xスレッド戦術

| 要素 | 推奨 |
|------|------|
| **長さ** | 5-10ツイート（5ツイートが最もRT/いいね比が高い） |

#### 0→10Kフォロワー成長戦術

### 2.2 TikTok FYPアルゴリズム — 2025-2026

#### コアメカニズム: 探索-活用システム（Explore-Exploit）
1. **初期配信プール**: 新動画 → 300-500人に配信
2. **評価指標**: 完視聴率 > 再視聴率 > シェア > コメント > いいね
3. **拡大判定**: 指標が閾値超え → 次のプール（5K → 50K → 500K → 全体）
4. **減衰**: 各プールで指標が落ちた時点で拡散停止

#### 最重要指標: 完視聴率（Watch-Through Rate）

| 動画尺 | 完視聴率の目安 | バイラル閾値 |
|--------|-------------|------------|
| 15秒以下 | 80%+ | 90%+ |
| 30秒 | 60%+ | 75%+ |
| 60秒 | 40%+ | 60%+ |
| 3分+ | 20%+ | 30%+ |

#### 最適動画長
- **Sweet Spot**: **30-60秒**（完視聴率と情報量のバランス最適）
- **教育系**: 45-90秒（1トピック1動画）
- **シリーズ物**: 各回60秒以内、3-5回シリーズ

#### 隠れ最強シグナル: 再視聴（Rewatch）
- 「もう1回見たい」→ 再生回数 > ユニーク視聴者数 → アルゴリズムが「質の高い」と判定
- **仕掛け方**: 画面の隅に伏線を仕込む / リスト型で「見逃し」を誘発 / 最後に冒頭の伏線を回収

#### TikTok SEO（検索エンジンとして）
- **Gen Zの40%がTikTokで検索**（Google代替）
- SEO重み順: 音声キーワード（最高）> 画面テキスト > キャプション > ハッシュタグ
- **実践**: 冒頭3秒で主要キーワードを「声に出す」+ テキストオーバーレイで表示

#### 教育系クリエイター向けプレイブック

| 要素 | 推奨 |
|------|------|
| **フック** | 0-3秒。Bold Statement / 数字 / 「あるある」型 |
| **フォーマット** | 顔出し + テキストオーバーレイ（ハイブリッド最強） |
| **編集** | ジャンプカット、3-5秒ごとにカット変更、無音排除 |
| **投稿頻度** | 週2-5回。火-木が最高エンゲージメント |
| **ハッシュタグ** | 3-5個（トレンド1-2 + ニッチ1-2 + ブランド1） |
| **シリーズ** | 番号付き3-5回シリーズ → ビンジ視聴を誘発 |
| **Duet/Stitch** | 大きいアカウントの誤情報を「修正」→ 権威構築 |
| **LIVE** | 週1回のQ&A → コミュニティ構築 |

#### 2026年1月 重要アップデート: Oracle移管
- **ByteDanceがUS事業をOracle主導のJVに移管**（2026年1月22日完了）
- **アルゴリズム再学習中**: US版はアメリカデータで再学習中（Q2 2026完了予定）
- 一部ユーザーで「古い動画ばかり表示される」現象 → 過渡期
- **日本への影響は限定的**だが、グローバルトレンドの波及に注意

### 2.3 YouTubeアルゴリズム — 2025-2026

#### コアモデル: 満足度加重型ディスカバリー（2026年シフト確定）

YouTube が公式に「**視聴者満足度調査が視聴時間よりも重い**」と確認。短い動画でも満足度が高ければ長尺に勝てる。

**ランキング方程式**（概念的）:
```
スコア = CTR × 視聴維持率 × 満足度調査結果 × パーソナライズ
```

#### 2026年の重要変更
- **Shorts収益化がLong-Formと同等に**: 一部地域でShortsの視聴時間あたり収益がLong-Formに追いつく
- **YPP参加要件緩和**: 500人+3本の公開動画+3,000時間（or Shorts 300万回/90日）
- **AI生成コンテンツの開示義務化**: ラベル未付与チャンネルは推薦が減少
- **「保存」が「いいね」より強いシグナルに**: DM共有もリーチを大幅ブースト

#### Shorts vs Long-Form

| 指標 | Shorts | Long-Form |
|------|--------|-----------|
| **目的** | 新規リーチ、トップオブファネル | 権威構築、ウォッチタイム蓄積 |
| **最適尺** | 30-60秒 | **8-15分**（教育系）/ 20-40分（深掘り） |
| **CTRベンチマーク** | 2-5%（フィード内） | 4-10%（検索/推薦） |
| **視聴維持率目安** | 70%+（バイラル閾値） | 50%+（平均23.7%） |
| **収益化** | CPM $0.04-0.10 | CPM $5-20（教育ニッチ） |
| **戦略的役割** | **リーチ拡大** | **収益+権威** |

#### サムネイル最適化の科学

| 原則 | データ根拠 |
|------|----------|
| **顔の表情**: 驚き > 好奇心 > ニュートラル | 表情ありサムネ = CTR 25-30%向上 |
| **テキスト**: 3語以内、太字、高コントラスト | 情報過多 → CTR低下 |
| **色彩**: メンタルヘルス系は落ち着いたトーン推奨 | 暖色/アンバー系がADHDブランドと一致 |
| **A/Bテスト**: YouTube内蔵機能で2案同時テスト | MrBeast: 1動画に20+サムネ候補を作成 |
| **顔の占有率**: 30-40%が最適 | 45,244動画の分析で「適度な顔表示」が最高エンゲージメント |

#### タイトル最適化

| 原則 | 例 |
|------|-----|
| **好奇心ギャップ** | 「ADHDの僕が○○したら、予想外のことが起きた」 |
| **数字 + 具体性** | 「ADHD脳の先延ばし方程式 — 3つの科学的解決策」 |
| **否定型フック** | 「ADHDは"治す"ものじゃない（精神科医も言わない理由）」 |
| **長さ** | 60文字以内（モバイル表示で途切れない） |

#### メンタルヘルス/ADHD YouTube 成長戦略

- **デュアルフォーマット**: 8-15分の教育動画（週1）+ Shorts（週2-3）
- **チャプター必須**: ADHD視聴者は「チャプターがあると見やすい」と明言
- **ハイブリッド映像**: 顔出し（30-40%）+ アニメーション/スライド（60-70%）
- **メンバーシップ**: 500人から開始可能。自助グループモデル（益田裕介方式）が有効
- **成長ベンチマーク**: 週2投稿で1,000人到達まで8-12ヶ月

## 3. バイラルフック心理学 & テンプレート

### 3.1 バイラルの科学

#### なぜシェアされるのか（Berger & Milkman, 2012）

| 感情 | シェア率への影響 | ADHD文脈での例 |
|------|----------------|--------------|
| **畏怖（Awe）** | +30% | 「ADHDの脳は1日に70,000の思考を処理する」 |
| **不安（Anxiety）** | +21% | 「ADHDの57%がローン返済を滞納している」 |
| **怒り（Anger）** | +34% | 「"普通にしろ"と言われ続けた30年」 |
| **実用的価値** | +25% | 「先延ばし方程式を30秒で説明する」 |
| **驚き（Surprise）** | +18% | 「ADHDは"治す"ものじゃない。理由は…」 |

#### バイラルフックの3大心理メカニズム

1. **パターン割込み（Pattern Interrupt）**: 予想を裏切り、注意を奪う
   - スクロール中の「え？」を作る
   - 逆説、矛盾、予想外の数字

2. **好奇心ギャップ（Curiosity Gap）**: 知りたいのに答えがない状態を作る
   - ツァイガルニク効果（未完了のタスクは記憶に残る）
   - 「答えは最後に」型の構成

3. **感情覚醒（Emotional Arousal）**: 高覚醒の感情はシェア行動を促進
   - 低覚醒（悲しみ、満足）→ シェアしない
   - 高覚醒（畏怖、怒り、不安、興奮）→ シェアする

### 3.2 バイラルフックテンプレート 20選

#### 逆説型（Pattern Interrupt）
1. 「ADHDは障害じゃない。○○だ」
2. 「先延ばしする人ほど、実は○○が高い」
3. 「"集中力がない"は大嘘。ADHDの真実は○○」
4. 「怠けてるんじゃない。脳の○○が違うだけ」

#### 数字型（Social Proof + Credibility）
5. 「ADHDの脳は1日○万の思考を処理する（定型の○倍）」
6. 「3ヶ月で○○を達成した、ADHDコーチングの全記録」
7. 「ADHD当事者の87%が"キャリアに悪影響"と回答した調査」
8. 「先延ばし方程式: モチベーション = (期待×価値) / (衝動性×遅延)」

#### あるある型（In-group Signaling）

## 7. ADHD当事者向けコンテンツデザイン原則

### 7.1 ADHD脳のコンテンツ消費パターン

| 特性 | コンテンツへの影響 | 対策 |
|------|------------------|------|
| **注意の持続困難** | 最初の3秒で判断される | 超強力なフック必須 |
| **ワーキングメモリ制約** | 長い説明は覚えられない | 1投稿1メッセージ |
| **ドーパミン報酬系** | 「面白い」→ ハイパーフォーカス | マイクロリワード（3-5秒ごと） |
| **時間感覚の歪み** | 長い動画 = 「1ヶ月かかる」 | チャプター + 短い動画 |
| **RSD（拒絶感受性）** | 批判的トーンで離脱 | 共感ファースト、「一緒に」の姿勢 |
| **夜型リズム** | ADHD成人の**80%が夜型**。メラトニン分泌が90分遅延 | **21-24時の投稿を重視** |
| **ドームスクロール** | SNS依存率: ADHD群 **15%** vs 対照群 3.3% | 「救い」になるコンテンツで差別化 |

**学術的裏付け**: 15研究・35,223人のメタ分析でADHD症状とSNS依存の中程度の正の相関（r=0.361）。ゲーム・テレビではなく**SNS固有**の現象（AAP縦断研究）。

### 7.2 テキストフォーマット（X, note）

- **短い段落**: 1-2文/段落（壁みたいなテキスト = 即離脱）
- **太字でスキャン可能に**: キーポイントを**太字**にする
- **箇条書き/番号リスト**: 長文より常にリスト形式
- **絵文字は控えめに**: 区切りとして使うのはOK、装飾過多はNG
- **空白行を多用**: 視覚的な「呼吸」を入れる

### 7.3 動画フォーマット（TikTok, YouTube）

- **カット頻度**: 3-5秒ごと（無音/静止画面 = 即スワイプ）
- **テキストオーバーレイ**: 音声と同時に表示（二重符号化理論）
- **BGM**: 10-20%音量（エネルギーは出すが邪魔にならない）
- **ズーム/パン**: キーポイントでズームイン
- **色の変化**: セクション切替時に背景色を変える

### 7.4 ADHD RSD配慮チェックリスト

コンテンツ作成時に必ず確認:
- [ ] 「普通の人と違って」→ NG（ノーマライゼーション違反）
- [ ] 「簡単にできる」→ NG（できない人を責める感覚）
- [ ] 「努力が足りない」→ NG（ADHD的に最悪のフレーム）
- [ ] 「みんなやってる」→ NG（孤立感を煽る）
- [ ] 「一緒にやってみよう」→ OK（仲間感）
- [ ] 「難しいよね、わかる」→ OK（共感ファースト）
- [ ] 「僕もこれで失敗した」→ OK（当事者性）

### 11.9 「科学的権威」ポジショニング評価

#### 方向性: ◎ 強く支持

1. **市場の空白**: 日本のADHDコンテンツは「困りごと」中心。「科学×強み」はブルーオーシャン
2. **信頼ギャップ活用**: TikTokのADHD動画の89%が科学的に不正確（Section 4参照）→ 科学ベースは最大の差別化
3. **匿名×科学の相性**: 「データが語る」アプローチが自然。匿名のほうが「人ではなく科学で語る」姿勢が強調される

#### 精緻化提案

1. **「科学の翻訳者」ポジション**: 研究者でなく「当事者目線で最新研究を届ける人」（How to ADHDモデル: "reliable curator and translator"）→「なぜ医師でないのに科学を語るのか」への事前回答
2. **「海外研究の日本語初解説」を看板に**: 英語論文→日本語は高い参入障壁（競合が真似しにくい）。「この研究、日本語で読めるのはここだけ」
3. **d値をブランドシンボルに**: 「d=0.89」をロゴ・バナー・投稿テンプレートに反復組込み → 「このアカウント=科学的」連想を形成
4. **エリート感を出しすぎない**: 科学用語は「＝平易な言い換え」を添える。「論文によると」→「最新の研究でわかったこと」。当事者の苦しみも必ず語る（共感なき権威は機能しない）
