### 離脱分析

- **平均視聴時間 ÷ 動画長** = 6.0秒 ÷ 36.58秒 = **16%** → 推定離脱ポイント: 約6秒
- **推定離脱ゾーン**: ビート2〜3の間（3-6秒）
- **離脱原因の仮説**: フック（ビート1「欠陥じゃない」）はスクロール停止に成功（847回表示）。しかし直後のビート2「ハイエンジン仕様」がメタファーに飛んでおり、「自分のことだ」と思わせる共感ビートが不足。96%が6秒以内に離脱。

### 学び

| 項目 | 内容 |
|------|------|
| **フック** | 逆説フック「欠陥じゃない」はスクロール停止に有効。847回表示は新規アカウント初投稿としては良好。 |
| **6秒の壁** | フック直後に共感/あるある（「自分のことだ」）を入れないと96%が離脱する。メタファーは6秒以降。 |
| **エンジンメタファー** | 繰り返し使用は有効だが、導入タイミングが早すぎた。先に共感を入れてからメタファーに移行すべき。 |
| **エスカレーション** | ビート5-7（強い→めちゃくちゃ強い→バケモン級→暴走）の畳みかけは効果的な構造。 |
| **パーソナル証拠** | 「ダメ営業から外資コンサルへ」は具体的で良い。 |
| **ビート粒度** | 1ビート=1文=最大15文字のルールはほぼ達成。ビート8・11がやや長い。 |
