## コンテンツ設計原則（ADHD視聴者向け）

1. **3秒ルール**: 3秒以内に判断される。フックは1.5秒で届ける
2. **1動画=1メッセージ**: ワーキングメモリ制約。詰め込みすぎない
3. **マイクロリワード**: 3-5秒ごとにテロップ切り替え/SE/色変化（ドーパミン報酬）
4. **二重符号化**: テロップ + 音声の二重で情報を入れる
5. **カット頻度**: 15-18カット/30秒（1.5-2秒間隔）
6. **BGM音量**: 10-20%（ナレーション最優先）

## 構造テンプレート（25-37秒）

```
[0:00-0:03] フック — スクロール停止（数字/逆説/共感）
[0:03-0:06] 追い打ち — フックを補強（比較/データ/疑問）
[0:06-0:10] 逆説/リフレーム — 常識をひっくり返す

[0:22-0:28] コアメッセージ — 1文で刺さる（ブランドタグライン活用）
[0:28-0:32] CTA — フォロー/コメント誘導
```

## 制作パイプライン

`/tiktok` Skillで以下の7フェーズを実行:

1. **Phase 0**: コンセプト定義（1メッセージ+認知変化+ソリューション設計）
2. **Phase 1**: フック開発（3案+採点→20パターン展開→選択）★最重要
3. **Phase 2**: 台本作成（テロップ+ナレーション+SE+映像指示）
4. **Phase 3**: ナレーション収録サポート
5. **Phase 4**: 動画生成（CapCut編集指示書）
6. **Phase 5**: 品質監査（RSD/法的/ファクトチェック）
7. **Phase 6**: 投稿最適化+振り返り
