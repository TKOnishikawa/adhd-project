### 収録のコツ

| ポイント | 説明 |
|---------|------|
| **テンポ** | やや速め（1.1-1.2倍速感覚）。ただしコアメッセージだけ1.3倍ゆっくり |
| **トーン** | 「友達に熱く語る」感じ。学者ではなく解説者。一人称「僕」 |
| **抑揚** | フック（強）→ 追い打ち（弱・優しい）→ リフレーム（強）→ 科学（堅→くだける）→ ソリューション（ドヤ）→ コア（優しい）→ CTA（フレンドリー） |
| **間** | セクション間に0.3-0.5秒の間。怖くて詰めすぎない |
| **感情のピーク** | 最低3回の起伏: ①フック「壊れてない」②リフレーム「怠けてるんじゃない」③コアメッセージ「知らないだけ」 |
| **NG** | 棒読み。早口すぎ。「えー」「あのー」のフィラー。 |

## 科学的根拠（ファクトチェック用）

| 主張 | 根拠 | 確度 |
|------|------|------|
| 先延ばしは怠けでなく脳の仕組みの違い | Steel (2007) 先延ばし方程式 / Barkley 実行機能モデル | 高（査読済み） |
| 実行機能 = 脳の「着火装置」 | Barkley EFモデル（実行機能＝行動の始動・抑制・調整） | 高（翻訳的表現） |
| ADHDの脳が動く5つのスイッチ（INCUP） | Dodson INCUP理論（Interest, Novelty, Challenge, Urgency, Passion） | 高（ADHDコーチング界で広く参照） |
| 「挑戦要素を足すと動ける」 | INCUPのChallenge要素。具体的な変化量は個人差あり | 中（「〜傾向がある」のニュアンス推奨） |

### 使用上の注意
- INCUPは「理論」として参照するが、「全員に必ず効く」などの断言はNG
- 「先延ばし方程式」の数値（式の係数）を具体的に引用しない（文脈によっては誤解を招く）
- ナレーション内の「〜と言われている」「〜傾向がある」を活用して断言を避ける

### 翻訳ブリッジ（専門用語 → 視聴者向け表現）

| 専門用語 | 台本内の表現 |
|---------|------------|
| EF（実行機能） | 脳の着火装置 |
| INCUP理論 | 脳の5つの着火スイッチ / 着火マップ |
| 先延ばし方程式 | やる気の公式（今回は直接言及せず） |
| 実行機能障害 | 着火装置の仕組みが違う |
