## 3. ポジショニング

### 3.1 差別化の3本柱

**1. 当事者性 × ハイパフォーマー経歴**

「ADHDの当事者がコーチ」は海外にはいるが、日本にはいない。さらに：
- アクセンチュアでマネージャー3年
- 25億スタートアップでDXチームマネージャー
- AI駆動開発で生産性3倍
- 外資ペットフードD2Cで売上YoY+35%（20億→27億）

→ **「ADHDでもハイパフォーマー」ではなく、「ADHDだからこそハイパフォーマー」** というナラティブ

**2. 海外最新論文ベース権威性（v2新規追加）**

TikTokのADHDコンテンツの50%がミスリーディング（UBC 2023研究）という状況で、全コンテンツに**論文レファレンスを標準装備**し、「科学的に信頼できる唯一の日本人ADHD発信者」として差別化する。

| プラットフォーム | 引用の出し方 |
|----------------|------------|
| TikTok / Shorts | テロップで「Source: {著者} et al., {年}」 |
| YouTube長尺 | 概要欄に全引用リスト + 画面にポップアップ |
| note | 記事末にReferences節（学術論文スタイル） |
| X | 「{大学名} {年} 研究」（1行で権威シグナル） |

**3. コミュニティ（6-8名の濃密グループ + バディ制度）**

- 大規模グループでも1on1でもない「6-8名」の黄金サイズ
- バディペア（毎朝5分チェックイン）
- ボディダブリング（週1回50分）
- LINEオープンチャット（RSD配慮ルール付き）

### 3.2 創業者パーソナルブランド設計

#### 西川太雄（Coach）— 「戦略家ADHD」

**SNSペルソナ**:
- **キャラクター**: データで語る戦略家。でもADHDの「衝動性」で行動力がある
- **トーン**: 知的だが親しみやすい。専門用語は必ず平易に言い換える
- **得意コンテンツ**: 論文解説×当事者体験、キャリア×ADHD、Before-After分析

**プロフィール案（v2更新）**:
> 🧠 ADHD当事者 | アクセンチュア→スタートアップDXマネージャー
> 📊 海外最新論文ベースの「ADHDの強み」| 効果量d=0.89の科学
> 🔥 100日で「脳の取扱説明書」を作るプログラム開発中
> 👇 ADHD Labs（LINEオープンチャット・無料）

#### 共同創業者（Engineer）— 「仕組み化ADHD」

**SNSペルソナ**:
- **キャラクター**: テクノロジーで「ADHD脳のハック」を実現する
- **トーン**: 実践的、ツール重視、「やってみた」系
- **得意コンテンツ**: ツールレビュー、環境デザイン、自動化Tips

## 4. プラットフォーム戦略

### 4.1 プラットフォームスコアリングと優先順位

v2では**ターゲット（30代+就労中ADHD当事者）へのリーチ × 信頼構築力 × 制作コスト**でスコアリングを実施し、プラットフォーム優先順位を全面見直し。

| 順位 | プラットフォーム | スコア | 役割 | 週間時間 |
|:----:|-----------------|:------:|------|---------|
| 最優先 | LINEオープンチャット | — | コミュニティプール | 3-4h |
| Tier 1 | note（週1記事） | 109/140 | 権威性構築 + SEO流入 | 4-6h |
| Tier 1 | YouTube長尺（月2本） | 113/140 | 深い信頼構築 + 検索流入 | 6-8h（月間） |
| Tier 2 | TikTok / YouTube Shorts | 78-82/140 | 認知拡大（切り抜き再利用） | 4-5h |
| Tier 3 | X | 82/140 | 専門家ネットワーク + 論文シェア | 2-3h |
| 削除 | Instagram | 75/140 | 優先度低（制作コスト高、ターゲット薄い） | — |

**合計**: 週19-26時間（2人で割って1人10-13時間/週）

> **ADHD創業者への自戒**: 「全部やる」は過集中→燃え尽きのパターン。4週時点で撤退基準未達のプラットフォームは**勇気を持って停止**する。note + TikTok + LINE OC の3本柱だけで十分戦える。

### 4.2 LINEオープンチャット詳細戦略

**Discordから変更した理由**:

| 比較項目 | Discord | LINEオープンチャット |
|---------|---------|------------------|
| 日本MAU | 800万（10-20代中心） | 累計3,000万+（全年代浸透） |
| 30代+での普及 | 激減 | 標準装備 |
| 参加ハードル | DL+登録+操作学習が必要 | タップ1回で参加 |
| 匿名参加 | 可能（設定による） | 可能（専用ニックネーム設定） |
| ADHD当事者の心理 | 「また新しいアプリ」障壁 | すでに使っている安心感 |

**ターゲット（30代+就労中ADHD当事者）にDiscordユーザーがほぼいない**という事実が最大の決定要因。

**コミュニティ名**: ADHD Labs（LINEオープンチャット）

**チャンネル（トーク）設計**:

| トーク | 目的 | 運用ルール |
|--------|------|-----------|
| #お知らせ | 運営情報・イベント告知 | 運営のみ投稿 |
| #自己紹介 | 新メンバー歓迎 | テンプレート形式（自由記入OK） |
| #もくもく | ボディダブリング告知・実況 | 参加自由、タスクシェア歓迎 |
| #論文ダイジェスト | 最新論文の要約投稿（週1） | 運営から発信、リアクション自由 |
| #つぶやき | 自由な思いつき・愚痴 | **返信不要文化**（ADHDの衝動性を肯定） |

**RSD配慮ルール（v1から継承）**:
- 「既読スルー」を責めない文化を明文化
- 批判的フィードバックは「サンドイッチ法」推奨
- 「参加しない自由」の保証
- 感情投稿への「共感リアクション」で応答（アドバイス禁止）

**LINE特有のリスク対策**:
- 荒らし対策: 管理者承認制（初期は設定推奨）
- 個人情報保護: 実名・連絡先をチャット内で共有しないルールを明文化
- スクリーンショット禁止明記（LINEのトーク内容は拡散リスクあり）

**ナッジBot**: LINE公式アカウントと連携したリッチメッセージ or Reminderアプリで実装（Discord Botは使用しない）

### 4.3 note 詳細戦略（Tier 1）

**2026年2月レコメンドエンジン刷新後の最新知見**:
- スキ数 > 読了率 > Google検索流入の重み付け
- レコメンド表示数**4.3倍**（2026年2月実測）
- 最適文字数: 2,000-5,000字（読了率最大化）

**無料80% / 有料20%の使い分け**:
- **無料**: あるある共感、基本Tips、体験談、論文解説（集客用）
- **有料（500-1,500円）**: 具体的フレームワーク、詳細ケーススタディ（プログラムの「味見」）

**SEOで狙うべきキーワード**:
- 「大人 ADHD 症状」「ADHD 仕事 できない」「ADHD タスク管理」
- 「ADHD コーチング 効果」「ADHD 強み」「大人 ADHD 対策」
- noteのドメインパワーが強く、Google検索上位に出やすい

**全記事共通: References節標準装備**:

```
## References
- [著者名] et al. (年). "[論文タイトル]." [ジャーナル名]. doi:XXXX
```

**投稿リズム**: 週1本（長文）。余裕があれば週2本目（短文Tips）

### 4.4 YouTube長尺 詳細戦略（Tier 1）

**v2新規追加。月2本に限定してクオリティ優先。**

**追加理由**:
- ¥30-50万の購入決断には「顔を見て、声を聞いて、深い知識を確認する」プロセスが不可避
- 概要欄からLP/LINEへの導線が自然
- SEOで「ADHD 大人 対策」等の検索から年単位で流入
- 参考: How to ADHD (Jessica McCabe) 190万フォロワー — 我々の目指すモデル

**2026年最新アルゴリズム知見**:
- CTR × 平均視聴時間の積が核心指標
- CTR目標: 5%以上（サムネイル最重要）
- 最初30秒で視聴者の40%を維持できれば成功
- 視聴時間より**満足度重視**にシフト（2026年確認済み）

**動画構成テンプレート（15-20分）**:

```
00:00-00:30 フック（「今日で、あなたのADHD観が変わります」）
00:30-02:00 今日学べること（アジェンダ）
02:00-12:00 本編（論文解説 × 当事者体験 × 実践Tips）
12:00-14:00 Q&A または事例紹介
14:00-15:00 CTA（LINE OC / ウェビナー登録）
```

**概要欄テンプレート**:

```
[概要説明 3-5行]

▼ 参照論文
- [引用1]
- [引用2]

▼ ADHD Labs（無料LINE OC）参加はこちら
[LINEオープンチャットURL]

▼ タイムスタンプ
[チャプター一覧]
```

**分担**: Coach = 顔出し（論文解説・当事者体験） / Engineer = 編集・サムネイル制作

**制作コスト軽減策**: TikTok/Shortsと同日収録。長尺本体 → Shortsに切り抜き再利用。

### 4.5 TikTok / YouTube Shorts 詳細戦略（Tier 2）

**v2新規追加（v1はShortsのみ、TikTok未記載）**

**TikTok追加理由**:
- 日本MAU 4,200万+
- ADHD当事者のプラットフォーム別利用率: TikTok > YouTube > IG >> X
- TikTokのADHDコンテンツ50%がミスリーディング（UBC 2023）→ 科学ベースで明確に差別化
- ただし¥30-50万の信頼構築は30秒では無理 → **認知の入口に徹する**

**重要**: 認知 → 信頼 → 購買のうち「認知」のみを担う。深い信頼構築はYouTube長尺とnoteの仕事。

**2026年最新アルゴリズム知見**:

| PF | 最重要指標 | 次点 | 最適尺 |
|----|-----------|------|--------|
| TikTok | リウォッチ率 | 完視聴率 > 保存率 | 15-30秒 |
| YouTube Shorts | 完視聴率 | リウォッチ > チャンネル登録 | 30-45秒 |

**TikTok特記事項**: 2026年1月にOracleへ移管完了。日本での継続利用は当面安定。

**最初3秒フック必須パターン**:
- 「ADHDの〇〇は嘘です（Source: [大学名]）」
- 「ADHDがあるのに月50万稼ぐ人の朝ルーティン」
- 「これを知ったらADHDの後悔がなくなる」

**同一素材の転用戦略**:

```
YouTube長尺収録
  └→ 核心30秒をカット
      ├→ TikTok用（縦型・BGM・テロップ調整）
      └→ YouTube Shorts（横→縦変換またはそのまま）
```

**6つの動画型（v1継承 + TikTok対応）**:
1. **あるある**: 「ADHDあるある: 3つのタスクを同時に始めて全部中途半端」
2. **論文速報**: 「[大学名]研究: ADHDの[テーマ]が[結果]だった件（Source表示）」
3. **Before-After**: 「Implementation Intentions導入前 vs 後」
4. **脳科学解説**: 「ドーパミンが足りないとこうなる（30秒で解説）」
5. **Tips速射**: 「ADHD的タスク管理3つのハック」
6. **Day in the life**: 「ADHDのDXマネージャーの1日」

### 4.6 X 詳細戦略（Tier 3）

**v2での位置づけ**: 最優先からTier 3に降格。30代+就労中ADHD当事者のX利用率が低いためターゲットリーチが弱い。ただし**医師・心理士・研究者との専門家ネットワーク**構築に有効なため継続。

**2026年最新アルゴリズム知見**:
- リプライ **27x** > ブックマーク **10x** > いいね **1x**（重み付け最新版）
- Premium加入で 25,000字まで投稿可能
- 投稿後の最初1時間のエンゲージメントが最重要

**週2-3時間の使い方**:
- 論文シェア（月 2-3本）
- 医師・研究者・心理士へのリプライ（週 5-10件）
- TikTok/noteのクロス告知
- X Communities参加（ADHD関連コミュニティ）

**コンテンツ比率（v1の40-30-20-10を維持）**:

| 分類 | 比率 | 目的 | 例 |

## 11. リスク管理

### 11.1 RSD（拒絶敏感性）対策（v1から継承）

**発信者としてのRSD防止プロトコル**:
1. **バッチ処理**: 投稿は予約投稿。リプライ確認は1日2回だけ（朝/夜）
2. **批判対応マニュアル**: 「事実に基づく反論」「共感で受ける」「スルー」の3分類を事前定義
3. **バディチェック**: 精神的にキツい批判を受けたら、共同創業者に即共有（1人で抱えない）
4. **24時間ルール**: 感情的な返信は24時間後に再判断

### 11.2 倫理・法的リスク（v1から継承）

- **コーチングと治療の境界**: 「治す」「改善する」ではなく「自己理解を深める」「仕組みを作る」と表現
- **景品表示法**: 効果を約束しない。「研究では効果量d=0.89」は事実の引用としてOK。「必ず改善します」はNG
- **個人情報**: 参加者の体験談は必ず書面許諾。匿名化オプション提供

### 11.3 ADHD創業者の「過集中→燃え尽き」防止（v1から継承）

- **週間上限**: 1人あたりSNS作業13時間/週（v2で2時間削減）
- **コンテンツバッファ**: 常に1週分の投稿を先行制作
- **月1外部チェック**: アドバイザー（ADHD非当事者）との1時間チェックイン

### 11.4 LINE OC特有のリスク（v2新規追加）

- **荒らし・不適切投稿**: 管理者承認制を初期設定推奨。違反者は即退会処理（管理者権限で実行可）
- **個人情報流出**: 電話番号・メールアドレスのチャット内共有を明示的に禁止ルール化
- **スクリーンショット拡散**: 利用規約に明記。「信頼の場」であることを繰り返し周知
- **なりすまし**: 公式アカウント（プロフィール画像+管理者バッジ）を明確化

### 11.5 TikTok特有のリスク（v2新規追加）

- **ミスリーディングコメント対応**: コメント欄に他ユーザーの誤情報が流れた場合、事実に基づいて冷静に訂正（削除より訂正を優先）
- **ネガティブバイラル対策**: 炎上防止のため、論文引用を徹底して「私の意見」ではなく「研究成果」として表現
- **Oracle移管後の規約変更**: 定期的に利用規約をチェック。代替としてYouTube Shortsの素材を常に保有

### 11.6 競合モニタリング（v1から継承）

- **月1回**: 競合6社（国内）+ 5社（海外）のSNS活動をチェック
- **アラート設定**: Google Alerts で「ADHD コーチング 日本」を監視
