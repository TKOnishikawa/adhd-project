# D. 行動科学フレームワーク ── ADHDコーチングの理論的基盤

> **調査日**: 2026-02-25
> **調査方法**: Gemini Search Grounding (deep_research + quick_search)
> **目的**: ADHDコーチングメソッド設計に活用できる行動科学フレームワークの体系的整理

---

## 1. テーマ概要

ADHDコーチングは「寄り添い」だけでは足りない。当事者の行動変容を**再現可能かつ科学的に支持された方法**で実現するには、行動科学の理論的枠組みが不可欠である。

本テーマでは、ADHDコーチングに適用可能な7つの主要フレームワークを取り上げる。いずれも一般心理学・行動科学で確立された理論であり、ADHD特有の神経生物学的特性（→ [Theme B: 神経科学的基盤](/adhd-project/research/B-neuroscience-mechanism)）を踏まえて応用することで、コーチングの効果を最大化できる。

各フレームワークは単独でも有用だが、**組み合わせて統合的に適用**することで最も効果を発揮する。例えば、ABC分析でトリガーを特定し、SDTで動機づけを設計し、ナッジで環境を整え、実行意図で行動を自動化する――という多層的アプローチである。

## 4. 各フレームワーク詳細

### 4.1 ABC分析（Antecedent-Behavior-Consequence）

#### 理論的背景

応用行動分析（ABA）の基本フレームワーク。行動を「先行因子（A）→ 行動（B）→ 結果（C）」の3要素で機能分析する。認知行動療法（CBT）の基盤技法でもある。

```
[A: 先行因子]          [B: 行動]           [C: 結果]
Antecedent             Behavior            Consequence
─────────────         ─────────           ──────────
何がトリガーか？        何をしたか？         何が起きたか？

例: 退屈な報告書       → SNSを開く         → 一時的な快感
    作業の中断指示     → 集中が完全に途切れ → タスク未完了
    朝の準備時間       → 過集中で出発遅延   → 遅刻・自己嫌悪
```

#### ADHDへの適用

ADHDの「問題行動」は意志の弱さではなく、**先行因子と結果の連鎖**として理解できる。

| ADHD文脈の先行因子 | 典型的な行動 | 結果（強化子） |
|-------------------|-------------|--------------|
| 退屈なタスク（低ドーパミン） | 先延ばし・回避 | 一時的な不快回避（負の強化） |
| 時間的プレッシャー（緊急性） | 過集中モードで猛ダッシュ | 締切ギリギリに完成（正の強化） |
| 刺激の多い環境 | 注意の逸脱 | 新奇な刺激による快感（正の強化） |
| 感情的な動揺 | 衝動的な発言・行動 | 感情の即時的発散（負の強化） |

<div class="coaching">

**コーチング実践**:
1. **A（先行因子）を変える**: 環境を整理する、作業前にルーティンを設定する
2. **B（行動）の代替を設計**: SNSの代わりに5分間散歩、衝動的発言の前に3秒ルール
3. **C（結果）を再設計**: 望ましい行動に即時報酬を設定（ポイント制、ご褒美）

</div>

<div class="tip">

**実践のコツ**: ABC分析シートをクライアントに渡し、1週間の「困った行動」を記録してもらう。コーチングセッションで一緒に分析すると、本人も気づかなかったパターンが見えてくる。

</div>

#### エビデンスと信頼性

- ABC分析はCBTの基盤技法として**極めて高い実証基盤**を持つ
- ABA由来であり、自閉スペクトラム症の行動介入で数十年の実績
- ADHD特異的なRCTは少ないが、CBT全体のエビデンスに包含される
- **信頼度: 高**

### 4.2 自己調整理論（Self-Regulation Theory）

#### Zimmermanの自己調整学習サイクル（2000）

教育心理学におけるメタ認知的自己調整の3フェーズモデル。学習だけでなく、あらゆる目標追求行動に適用可能。

```
     ┌──────────────────────────────────────┐
     │                                      │
     ▼                                      │
[Phase 1: 予見]     →    [Phase 2: 遂行]    →    [Phase 3: 自己反省]
 ・目標設定              ・注意集中              ・自己評価
 ・戦略計画              ・自己教示              ・帰属（原因分析）
 ・自己効力感            ・自己モニタリング       ・自己反応（満足/不満）
 ・タスク分析            ・イメージング          ・適応的判断
     ▲                                      │
     │                                      │
     └──────────────────────────────────────┘
              （サイクルとして繰り返し）
```

#### ADHDにおける自己調整の困難

ADHDでは3フェーズ**すべて**に困難が生じる：

| フェーズ | 定型発達 | ADHD | 神経科学的背景 |
|---------|---------|------|--------------|
| 予見 | 適切な目標設定・計画 | 過大な目標、計画の欠如 | PFC（前頭前野）の機能不全 |
| 遂行 | 自己モニタリング | 注意の逸脱、過集中 | DMN-TPN切り替え不全 |
| 自己反省 | 原因の客観的分析 | 全否定/楽観的すぎ | 情動調整の困難 |

#### Baumeisterの自我消耗モデル（Ego Depletion）

- **元来の主張**: 意志力は有限資源であり、使うと消耗する
- **近年の議論**: 再現性に深刻な疑問（2016年マルチラボ・レプリケーションで非有意）
- **現在の位置づけ**: 「消耗」は信念・期待の影響が大きい可能性（限定的資源モデルの修正版）

<div class="pitfall">

**注意**: 自我消耗を理由に「意志力が尽きたから仕方ない」と正当化するのは科学的に支持されない。ただし、ADHDの当事者にとって**認知的負荷が高い状態が続くと遂行能力が落ちる**という経験則は臨床的に妥当であり、休息・回復の設計は重要。

</div>

<div class="coaching">

**コーチング実践**:
1. **予見フェーズの強化**: コーチと一緒に目標を現実的なサイズに分解。「If-Then計画」の事前設定
2. **遂行フェーズの外部化**: タイマー、チェックリスト、コーチとのチェックイン（毎日5分）
3. **自己反省フェーズの構造化**: 「できたこと3つ」リスト → 全否定パターンの回避

</div>

#### エビデンスと信頼性

- Zimmermanモデルは教育心理学で**広く実証**されている
- ADHD文脈では自己調整介入のメタ分析が複数存在（中〜高効果量）
- Baumeisterの自我消耗はレプリケーション問題あり（要注意）
- **信頼度: 高（Zimmerman）/ 低〜中（Baumeister自我消耗）**

### 4.3 SDT: 自己決定理論（Self-Determination Theory）

#### 理論的背景（Deci & Ryan, 2000/2008）

人間の内発的動機づけは3つの基本的心理欲求（Basic Psychological Needs）の充足度に依存する。

| 基本欲求 | 説明 | ADHDで不足しやすい理由 |
|---------|------|---------------------|
| **自律性（Autonomy）** | 自分で選び、行動をコントロールしている感覚 | 「やらされ感」の強い環境、構造化への抵抗 |
| **有能感（Competence）** | 効果的に行動できている感覚 | 繰り返す失敗、比較による自己評価低下 |
| **関係性（Relatedness）** | 他者とつながり、受け入れられている感覚 | 社会的スキルの困難、RSD（拒絶感受性） |

#### 動機づけの連続体（SDTスペクトラム）

```
[外発的動機づけ]                                              [内発的動機づけ]
 ├── 外的調整 ── 取入れ的調整 ── 同一化的調整 ── 統合的調整 ──── 内発的動機 ──┤
 │   「罰を避ける」「恥を避ける」「価値を認識」「自己と統合」   「面白いから」 │
 │                                                                         │
 │   ← ADHDで陥りやすい範囲 →              ← コーチングで目指す範囲 →      │
```

#### ADHDコーチングへの適用

ADHD当事者は幼少期から「できない」フィードバックを受け続け、3欲求が慢性的に充足されにくい。コーチングでは意図的に3欲求を充足する設計が必要。

| 基本欲求 | コーチングでの充足方法 | 具体例 |
|---------|---------------------|-------|
| 自律性 | 選択肢の提示、進め方はクライアントが決定 | 「A案とB案、どちらで進める？」 |
| 有能感 | 小さな成功の積み上げ、スモールステップ | 「先週の3つの達成を振り返ろう」 |
| 関係性 | 無条件の肯定的関心、共感的傾聴 | 「失敗しても、あなたの価値は変わらない」 |

<div class="coaching">

**コーチング実践**:
- **セッション設計**: 毎回、3欲求のうち少なくとも1つを意図的に充足するアクティビティを含める
- **自律性の保護**: 「〜すべき」ではなく「〜したいとしたら、どんな方法がある？」
- **有能感の構築**: 「できなかったこと」より先に「できたこと」を確認
- **関係性の基盤**: コーチとの安全な関係が、外部の対人関係改善のモデルになる

</div>

<div class="tip">

**実践のコツ**: ADHDクライアントは「やるべきことをやれない自分」に強い罪悪感を抱いていることが多い。SDTの視点で「動機づけの種類が合っていないだけ」とリフレーミングするだけで、大きな安堵感を得られることがある。

</div>

#### エビデンスと信頼性

- SDTは動機づけ研究の**最も確立された理論**の一つ（数千本の研究論文）
- 教育、スポーツ、医療、職場など広範な文脈で実証済み
- ADHD特異的な適用研究は増加中だが、まだコーチング文脈のRCTは少ない
- **信頼度: 高（理論自体）/ 中（ADHD特異的適用）**

### 4.4 CBT Safrenモデル（認知行動療法 for ADHD）

#### 理論的背景（Safren et al., 2005）

Steven Safrenがマサチューセッツ総合病院で開発した、**成人ADHD専用の構造化CBTプログラム**。薬物療法で十分な改善が得られない成人ADHDに対する心理的介入として設計された。

#### 4つのコアモジュール

| モジュール | 目的 | 主な技法 | セッション数 |
|-----------|------|---------|-------------|
| **1. 組織化と計画** | 日常生活の構造化 | タスクリスト、カレンダー、優先順位づけ、問題解決スキル | 5回 |
| **2. 不注意への対処** | 注意散漫の管理 | 気づきの練習、注意の再集中技法、環境調整 | 3回 |
| **3. 適応的思考** | 否定的自動思考の修正 | 認知的再構成、思考記録、証拠の検討 | 3回 |
| **4. 先延ばしの克服** | 回避行動の解消 | 行動活性化、タスク分割、報酬設計 | 2回 |

#### ADHD特有の認知的歪み（Safrenが指摘）

ADHDの当事者に特徴的な否定的自動思考のパターン：

```
「また失敗した。自分はダメな人間だ」          → 全か無か思考
「他の人は普通にできるのに、なぜ自分だけ」      → 比較と過度の一般化
「どうせ計画を立てても、守れないに決まっている」  → 破局的思考・占い的思考
「この前うまくいったのは、たまたまだ」          → ポジティブの無視
「こんな簡単なこともできないなんて」            → すべき思考
```

#### RCTの結果（Safren et al., 2005, JAMA）

| 指標 | CBT群 | リラクゼーション+教育群 | 効果量 |
|------|-------|---------------------|--------|
| ADHD症状改善 | 有意に改善 | 改善なし | **d = 0.89（大）** |
| 不安症状改善 | 有意に改善 | 改善なし | d = 0.67（中〜大） |
| 抑うつ症状改善 | 有意に改善 | 改善なし | d = 0.61（中〜大） |
| CGI改善評価 | 56%が改善 | 13%が改善 | ― |

<div class="coaching">

**コーチングへの転用**:
- Safrenモデルは**臨床家（心理士・精神科医）向け**に設計されたプログラムであり、コーチが「治療」として実施するものではない
- ただし、以下の要素はコーチングに転用可能：
  1. **組織化・計画スキル**: タスクリスト、カレンダーの使い方
  2. **認知的歪みへの気づき**: 「その考え、本当？」の問いかけ
  3. **行動活性化の原則**: 小さな一歩から始める設計
- **臨床と非臨床の線引き**: 認知的再構成を深く掘り下げる場合は専門家への紹介が必要

</div>

<div class="pitfall">

**注意**: コーチは「治療」を行うわけではない。Safrenモデルの組織化・計画スキル部分はコーチングと親和性が高いが、認知的再構成の深い部分は臨床心理士の領域。クライアントの状態を見て、適切に専門家へ紹介する判断力が求められる。

</div>

#### エビデンスと信頼性

- JAMAに掲載されたRCTで**大きな効果量**が確認された最もエビデンスの強いADHD心理介入
- 後続のRCTでも一貫した結果（Solanto et al., 2010も同様の効果量）
- 薬物療法との併用で最大の効果
- **信頼度: 高（RCT複数、メタ分析あり）**

### 4.5 ACT: アクセプタンス&コミットメント・セラピー

#### 理論的背景

第三世代CBT（認知行動療法の発展形）。問題となる思考や感情を「変える」のではなく、**「受け入れた上で」価値に沿った行動を選ぶ**ことを目指す。

#### ACTの6つのコアプロセス（ヘキサフレックス）

```
                    心理的柔軟性
                   （中心目標）
                  /      |      \
        受容    脱フュージョン  今この瞬間への注意
        /              |                \
  価値の明確化    コミットされた行動    文脈としての自己
```

| プロセス | 説明 | ADHD文脈での意味 |
|---------|------|----------------|
| **受容** | 不快な思考・感情に抵抗しない | 「集中できない自分」との闘いをやめる |
| **脱フュージョン** | 思考を「事実」ではなく「心の出来事」と見る | 「自分はダメだ」→「自分はダメだという考えが浮かんだ」 |
| **今この瞬間** | 過去の後悔・未来の不安から離れ、今に注意を向ける | マインドワンダリング対策 |
| **文脈としての自己** | 観察者としての自己 | ADHDラベルに同一化しすぎない |
| **価値の明確化** | 何が本当に大切かを明らかにする | 外部的「すべき」ではなく、自分の価値で行動を選ぶ |
| **コミットされた行動** | 価値に沿った具体的な行動を選び実行する | 完璧でなくてよい、価値の方向に一歩踏み出す |

#### ADHDへの特異的適用

ADHDの当事者は**経験的回避**（嫌な思考・感情を避ける行動パターン）に陥りやすい：

- 退屈なタスク → 回避（先延ばし）→ 罪悪感 → さらなる回避 → 悪循環
- 過去の失敗記憶 → 回避（新しい挑戦をしない）→ 機会損失

ACTは**回避ではなく受容**を促し、不完全でも価値に沿った行動を選択する力を育てる。

#### RCTの結果（Livheim et al., 2014他）

| 研究 | 対象 | 結果 | 効果量 |
|------|------|------|--------|
| Livheim et al. (2014) | 成人ADHD | ADHD症状・QOL有意改善 | **d = 0.47〜0.62（中）** |
| Pettersson et al. (2014) | 成人ADHD（インターネットACT） | 症状改善・心理的柔軟性向上 | d = 0.40（中） |

<div class="coaching">

**コーチング実践**:
1. **価値の発見ワーク**: 「もしADHDが魔法で消えたとしても変わらない、あなたが大切にしたいことは？」
2. **脱フュージョンのエクササイズ**: 否定的思考を「パッセンジャー（乗客）」に例える。バスの運転手（自分）は乗客の叫びに左右されず、行きたい方向に進む
3. **コミットメントの小さな一歩**: 「今週、価値に沿った小さな行動を1つだけ選ぶとしたら？」
4. **受容の実践**: 「集中できない時間がある。それはADHDの特性であり、自分の価値を左右しない」

</div>

#### エビデンスと信頼性

- ACTは不安・うつ・慢性疼痛など広範な領域で確立（メタ分析多数）
- ADHD特異的なRCTは増加中。効果量は中程度
- Safren CBTほどの大効果量ではないが、**受容ベースのアプローチ**として独自の価値がある
- **信頼度: 中〜高**

### 4.6 習慣形成科学（Habit Formation）

#### 理論的背景

習慣は「キュー → ルーチン → 報酬」のループで形成される（Duhigg, 2012; Wood & Neal, 2007）。一度形成された習慣はワーキングメモリを消費せず自動的に実行されるため、**ADHDの実行機能障害を迂回する強力な戦略**となる。

#### 習慣ループ

```
    ┌─────────────┐
    │  キュー(Cue) │ ← 時間、場所、感情、直前の行動、人
    └──────┬──────┘
           ▼
    ┌──────────────────┐
    │ ルーチン(Routine) │ ← 自動化された行動パターン
    └──────┬───────────┘
           ▼
    ┌──────────────┐
    │ 報酬(Reward) │ ← ドーパミン放出 → ループの強化
    └──────┬───────┘
           │
           └──→ キューとルーチンの結合を強化
```

#### 実行意図（Implementation Intentions）── Gollwitzer (1999)

「いつ、どこで、何をするか」を事前に具体的に決めておく手法。**If-Thenプランニング**とも呼ばれる。

```
一般的な目標意図:  「運動をもっとする」
実行意図:         「月曜と水曜の朝7時に、玄関で靴を履いて、20分歩く」

一般的な目標意図:  「先延ばしを減らす」
実行意図:         「もし退屈を感じたら、5分だけタスクに取り組む（5分ルール）」
```

#### メタ分析の結果（Gollwitzer & Sheeran, 2006）

- **94の独立研究のメタ分析**: 実行意図は目標達成に**中〜大の効果量（d = 0.65）**
- 特に**習慣的な行動の変更**において効果が大きい
- ADHDの「やるつもりなのにできない」問題に直接対処

#### ADHDへの特異的適用

| ADHD特有の困難 | 習慣形成での対処 | 具体例 |
|--------------|----------------|-------|
| ワーキングメモリ不足 | 習慣化でWM不使用に | 「鍵は必ず玄関のフックに掛ける」 |
| 実行機能の弱さ | If-Thenで意思決定を不要に | 「朝起きたら→まず水を飲む」 |
| 動機づけの変動 | キューに結びつけて自動発火 | 「コーヒーを入れたら→薬を飲む」 |
| 時間盲（Time Blindness） | 時間ではなく行動をキューに | 「歯を磨いたら→荷物チェック」 |

<div class="coaching">

**コーチング実践**:
1. **キュー（トリガー）の設計**: 既存の習慣にスタック（紐づけ）する。「〇〇した直後に△△」
2. **2分ルール**: 新しい習慣は最初「2分以内」の極小バージョンから始める
3. **報酬の即時化**: 習慣実行直後に小さな報酬（チェックマーク、好きな曲1曲）
4. **環境の事前準備**: 翌日の服を前夜に出す、サプリを目に見える場所に置く
5. **リセットの許容**: 1日途切れても「失敗」ではなく「次のキューで再開」

</div>

<div class="tip">

**実践のコツ**: ADHDの当事者にとって「毎日同じことを同じ順番で」は最も退屈に感じるパターン。ルーチンの**核心部分は固定**しつつ、周辺に**変化・新奇性**を入れるとドーパミン系を味方にできる。例：朝のルーチンの音楽は毎回変える、歩くコースは3パターン用意する。

</div>

#### エビデンスと信頼性

- 習慣形成のメカニズムは行動科学で十分に実証されている
- 実行意図のメタ分析は強力なエビデンス（94研究、d = 0.65）
- ADHD特異的な習慣形成介入のRCTは少ないが、原理の適用は論理的に妥当
- **信頼度: 高（実行意図）/ 中〜高（ADHD文脈での適用）**

### 4.7 ナッジ理論（Nudge Theory）

#### 理論的背景（Thaler & Sunstein, 2008）

行動経済学のフレームワーク。人の行動を**禁止や命令ではなく、選択肢の設計（チョイスアーキテクチャ）**によって望ましい方向に促す。

#### ナッジの5原則（ADHDコーチング版）

| 原則 | 説明 | ADHD文脈での適用 |
|------|------|----------------|
| **デフォルト変更** | 何もしなければ望ましい結果になるよう設計 | スマホの通知を全オフがデフォルト |
| **努力の削減** | 望ましい行動のフリクション（摩擦）を減らす | 薬を洗面台に置く、服を前日に用意 |
| **努力の増大** | 望ましくない行動のフリクション増大 | SNSアプリを奥のフォルダに移動 |
| **社会的規範** | 他者の行動を参照点にする | 「ADHDコミュニティの80%がこの方法を使っている」 |
| **リマインダー** | タイミングよく促す | 時間帯ごとのスマートリマインダー |

#### チョイスアーキテクチャの実例

```
【Before: 努力依存の設計】
  朝の出発準備:
  1. 何を持っていくか思い出す（WM負荷）
  2. それぞれの場所から持ってくる（実行機能負荷）
  3. 忘れ物がないか確認する（注意力負荷）
  → 高確率で忘れ物、遅刻

【After: ナッジ設計】
  朝の出発準備:
  1. 「出発セット」を玄関に常設（デフォルト変更）
  2. 鍵・財布・スマホの定位置を色テープで明示（努力の削減）
  3. ドアに「3点チェック」の貼り紙（リマインダー）
  4. 出発15分前のスマートスピーカー通知（リマインダー）
  → WM・実行機能を使わず自動的に準備完了
```

<div class="coaching">

**コーチング実践**:
1. **環境監査**: クライアントの生活環境を一緒に見直し、「努力しなくても良い行動が起こる」設計を作る
2. **デジタルナッジ**: スマホのホーム画面設計（仕事アプリを1画面目、SNSを3画面目に）
3. **デフォルト設定**: カレンダーの予定にバッファ時間をデフォルトで入れる
4. **摩擦の増減**: ゲームのコントローラーを引き出しの奥に → 「ちょっとだけ」の誘惑を減らす

</div>

<div class="tip">

**実践のコツ**: ナッジは「気合い」や「意志力」に頼らない点がADHDに最適。環境を変えれば行動が変わる。クライアントに「環境を変えることは、ズルではなく賢さだ」と伝えると、罪悪感なくナッジを受け入れられることが多い。

</div>

#### エビデンスと信頼性

- ナッジ理論は行動経済学の主要理論（セイラーはノーベル経済学賞受賞）
- 公衆衛生、金融行動、環境行動で大規模実証あり
- ADHD特異的なナッジ介入のRCTは未実施だが、原理の適用は論理的に極めて妥当
- → [Theme B: 神経科学的基盤](/adhd-project/research/B-neuroscience-mechanism)の「環境設計原則」と直接接続
- **信頼度: 高（理論自体）/ 中（ADHD特異的適用）**

## 5. フレームワーク間の統合アプローチ

7つのフレームワークは独立したものではなく、**多層的に組み合わせることで最も効果を発揮**する。

### 統合モデル: ADHDコーチングの4層構造

```
Layer 4: 認知・感情層（CBT Safren + ACT）
  └── 否定的自動思考の修正 + 受容 + 価値に沿った行動

Layer 3: 動機づけ層（SDT）
  └── 自律性・有能感・関係性の充足設計

Layer 2: 行動自動化層（習慣形成 + 実行意図）
  └── If-Thenプランニング + ルーチン構築

Layer 1: 環境層（ナッジ + ABC分析）
  └── トリガーの特定 + チョイスアーキテクチャ

基盤: 自己調整サイクル（Zimmerman）
  └── 予見 → 遂行 → 自己反省 のメタ認知フレーム
```

### 統合の実践例: 「朝の出発準備」改善プロジェクト

| 層 | フレームワーク | 適用内容 |
|----|-------------|---------|
| 環境層 | ABC分析 | 「朝の時間帯」×「SNS」→「遅刻」のパターンを特定 |
| 環境層 | ナッジ | 出発セットの常設、スマホのSNS通知オフ |
| 行動層 | 習慣形成 | 「歯を磨いたら → 荷物チェック」のスタッキング |
| 行動層 | 実行意図 | 「もしSNSを開きたくなったら → 5分間散歩」 |
| 動機づけ層 | SDT | 準備方法はクライアント自身が選択（自律性） |
| 認知層 | ACT | 「また遅刻しそう」という思考を脱フュージョン |
| 認知層 | CBT | 「いつも遅刻する」→「先週は3回中2回間に合った」 |
| メタ認知 | 自己調整 | 週末に1週間を振り返り → 翌週の計画を調整 |

| クライアントの状態 | 最優先 | 次の段階 | 理由 |

## 9. 注意点・限界

### 臨床とコーチングの境界

- CBT Safrenモデル、ACTは元来**臨床的介入**であり、深い認知的再構成はコーチの範囲外
- コーチが活用できるのは「スキル訓練」「心理教育」「受容の態度」の部分
- クライアントの状態が臨床レベル（うつ、自傷、解離等）であれば**必ず専門家に紹介**

### エビデンスの限界

- ADHD特異的なRCTが存在するのはCBT Safren、ACT、実行意図の3つのみ
- SDT、ナッジ、ABC分析は一般的なエビデンスは豊富だが、ADHD文脈での専用RCTは少ない
- 「ADHDコーチング」自体のRCTが国際的にも極めて少ない（→ Theme E で詳述予定）

### 文化的コンテキスト

- 主要研究は欧米（特に北米）で実施されており、**日本の文化的コンテキスト**での妥当性は未検証
- 自律性の重視度、関係性の質、「受容」の概念は文化によって異なる可能性
- 日本の当事者に適用する際は、文化的適応が必要

### 個人差への注意

- ADHDは異質性の高い状態であり、全員に同じフレームワークが効くわけではない
- ソヌガ＝バークの三重経路モデル（→ [Theme B](/adhd-project/research/B-neuroscience-mechanism)）に基づき、クライアントの主要経路に応じたフレームワーク選択が重要
