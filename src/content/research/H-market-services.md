# H. ADHD コーチング市場・既存サービス分析

> **調査日**: 2026-02-25
> **調査方法**: Gemini Search Grounding (deep_research + quick_search)
> **目的**: ADHDコーチングの国内外市場を俯瞰し、サービス設計における差別化機会を特定する

---

## 1. テーマ概要

ADHDコーチング市場は、(1) 国際的な専門コーチング組織の整備、(2) デジタルセルフヘルプツールの急成長、(3) 日本市場の独自構造、の3つの観点から理解する必要がある。

日本市場は欧米と比較して以下の構造的特徴を持つ：

- **成人ADHD支援の制度的空白**: 児童向け支援は充実しているが、成人向けは極めて限定的
- **専門コーチの不在**: ADHDコーチングの国際認定資格を持つ日本語対応コーチがほぼ存在しない
- **グレーゾーン層の受け皿不足**: 確定診断に至らないが困難を抱える層へのサービスが欠如
- **心理的支援の統合不足**: 就労支援・スキル訓練は存在するが、自己肯定感・アイデンティティ再構築への包括的アプローチが不足

## 3. 国際コーチング組織分析

### 3.1 ADDCA（ADD Coach Academy）

ADHDコーチングに特化した世界最大のトレーニング機関。

| 項目 | 内容 |
|------|------|
| 設立 | 1998年 |
| 認定 | ICF認定ADHDコーチトレーニングプログラム（ACTP） |
| 特徴 | ADHDの神経科学に基づくコーチング手法 |
| 対象 | コーチ養成（クライアント直接支援ではない） |
| 日本語対応 | なし（英語のみ） |

<div class="coaching">

**コーチングへの示唆**: ADDCAのカリキュラムは、ADHDコーチに必要な専門知識の標準を定義している。日本で同等の養成を行う機関は存在しない。

</div>

### 3.2 ICF（International Coaching Federation）

コーチング業界全体の国際認定機関。ADHDに特化はしていないが、コーチの質の担保基準を設定。

| 項目 | 内容 |
|------|------|
| 認定レベル | ACC, PCC, MCC（累積時間で昇格） |
| ADHDとの関係 | ICF認定 + ADHDコーチング専門スキルの組み合わせが理想 |
| 日本の状況 | ICF認定コーチは増加傾向だが、ADHD専門性との組み合わせはごく少数 |

### 3.3 CHADD（Children and Adults with ADHD）

米国最大のADHD当事者・家族支援組織。

| 項目 | 内容 |
|------|------|
| 設立 | 1987年 |
| 役割 | 教育、啓発、アドボカシー（政策提言） |
| リソース | 専門家ディレクトリ、ウェビナー、支援グループ |
| コーチングとの関係 | コーチング推奨、専門コーチ紹介あり |
| 日本語対応 | なし |

### 3.4 ACO（ADHD Coaches Organization）

ADHDコーチの専門職組織。

| 項目 | 内容 |
|------|------|
| 役割 | ADHDコーチの倫理基準・専門性基準の策定 |
| 認定 | ADHDコーチの専門認定あり |
| ディレクトリ | ADHDコーチ検索サービスを提供 |
| 日本語対応コーチ | 極めて限定的（ほぼゼロ） |

<div class="pitfall">

**重要な注意**: これらの国際組織は英語圏を中心に発展しており、日本語でサービスを受けられるADHDコーチは国際認定を持つ人でもほぼ存在しない。これは「市場の空白」であると同時に「参入障壁が低い」ことも意味する（品質基準が未整備）。

</div>

## 4. デジタルツール比較表

ADHDに特化したデジタルツールは近年急速に増加している。主要なものを以下に比較する。

### 4.1 ツール一覧

| ツール | 種別 | 特徴 | 日本語対応 | 価格帯 | 対象 |
|--------|------|------|-----------|--------|------|
| **Tiimo** | 視覚プランナー | 視覚的な日課管理、スケジュール | あり | 無料〜$6.99/月 | ADHD/自閉症 |
| **Inflow** | CBTベースアプリ | 認知行動療法プログラム、AI「Quinn」 | なし（英語） | $9.99/月〜 | 成人ADHD |
| **Numo** | ゲーミフィケーション | タスク管理＋コミュニティ＋ゲーム要素 | なし（英語） | 無料〜有料 | 成人ADHD |
| **Focusmate** | バーチャルコワーキング | 25/50分セッション、相互監視 | UI英語（利用は言語不問） | 無料〜$6.99/月 | 集中困難全般 |
| **Habitica** | RPGタスク管理 | タスク完了でキャラ成長 | 日本語あり | 無料〜 | ゲーム好き全般 |
| **Forest** | 集中タイマー | 集中時間で木が育つ | 日本語あり | 買い切り ¥490 | 集中困難全般 |

### 4.2 主要ツール詳細

#### Tiimo（ティーモ）

デンマーク発。ADHD・自閉症の当事者チームが開発した視覚プランニングアプリ。

- **強み**: 視覚的なタイムラインUI、カスタマイズ可能なルーティン、Apple Watch対応
- **ADHDとの親和性**: 時間盲（Time Blindness）に対応する視覚的フィードバック
- **限界**: あくまでプランニングツール。心理的サポートやコーチング機能はない

<div class="coaching">

**コーチングへの示唆**: Tiimoのような視覚ツールはコーチングの**補完ツール**として活用可能。コーチがクライアントと一緒にルーティンを設計し、Tiimoで実装・モニタリングする運用が考えられる。

</div>

#### Inflow（インフロー）

CBT（認知行動療法）ベースのADHD特化アプリ。AI「Quinn」によるパーソナライズドサポートが特徴。

- **強み**: エビデンスベースのCBTプログラム、AIチャットボット、進捗トラッキング
- **ADHDとの親和性**: 構造化された学習モジュール、即時フィードバック
- **限界**: 英語のみ、日本の保険制度・文化への対応なし

<div class="tip">

**差別化のヒント**: Inflowの「AIコーチ」は知識提供に強いが、**人間のコーチにしかできない「共感」「承認」「アイデンティティ支援」**は自動化できない。これが人間コーチの不可替性を示す。

</div>

#### Numo（ヌモ）

ゲーミフィケーションとコミュニティを軸にしたADHD支援アプリ。

- **強み**: 楽しさ駆動のタスク管理、仲間との繋がり、ゲーム要素による動機づけ
- **ADHDとの親和性**: 報酬系の即時化（ドーパミン仮説に対応）
- **限界**: 英語のみ、ゲーム要素への飽きの可能性

## 5. 日本固有サービスの詳細分析

### 5.1 リタリコ（LITALICO）

発達障害支援の最大手。上場企業。

| 項目 | 内容 |
|------|------|
| 主要サービス | リタリコワンダー（児童IT教育）、リタリコジュニア（児童発達支援）、リタリコワークス（就労移行支援） |
| 対象 | **児童中心**。成人はワークスのみ |
| 価格 | 授業料 約22,000円/回（ワンダー）、自立支援給付利用で自己負担軽減（ワークス） |
| 強み | 全国規模、知名度、上場の信頼性 |
| ADHD対応 | ADHDに特化はしていない（発達障害全般） |
| コーチング要素 | なし（療育・就労スキル訓練が中心） |

<div class="pitfall">

**構造的限界**: リタリコは「児童の療育」と「成人の就労支援」がコア事業。成人ADHDの**日常生活・心理的課題・キャリアデザイン**へのアプローチは事業範囲外。

</div>

### 5.2 Kaien（カイエン）

発達障害に特化した就労支援のパイオニア。

| 項目 | 内容 |
|------|------|
| 主要サービス | 就労移行支援事業所、ガクプロ（大学生向け） |
| 対象 | 成人（就労困難者）、大学生 |
| 価格 | **約90%が自己負担0円**（障害福祉サービスの枠組み） |
| 強み | 発達障害専門の就労支援ノウハウ、高い定着率 |
| ADHD対応 | ADHDを含む発達障害全般に対応 |
| コーチング要素 | 限定的（就労スキル訓練・マッチングが中心） |

<div class="coaching">

**コーチングへの示唆**: Kaienは就労という「出口」に強い。しかし、就職後の**継続的なセルフマネジメント支援**や**職場での適応コーチング**は制度上カバーされない。ここが「就労後コーチング」のニーズゾーン。

</div>

### 5.3 発達障害者支援センター（公的機関）

各都道府県に設置された発達障害支援の公的相談窓口。

| 項目 | 内容 |
|------|------|
| 設置 | 発達障害者支援法に基づく |
| 対象 | 全年齢の発達障害当事者・家族 |
| 価格 | **無料** |
| 強み | 公的機関の安心感、医療・福祉の橋渡し |
| 限界 | 相談待機が長い（数ヶ月〜）、個別継続支援は困難、地域差大 |
| コーチング要素 | なし（情報提供・相談・紹介が中心） |

### 5.4 ホスピタリッチ

成人ADHDに特化したコーチングを提供する数少ない日本語サービス。

| 項目 | 内容 |
|------|------|
| 形態 | オンラインコーチング |
| 対象 | **成人ADHD**（当事者および疑い） |
| 特徴 | ADHDに特化、当事者理解に基づくコーチング |
| 価格 | 個別見積もり（非公開） |
| 限界 | 小規模、認知度低、標準化された手法の開示なし |

<div class="tip">

**市場ポジション**: ホスピタリッチは「成人ADHD × コーチング × 日本語」の数少ない直接競合。しかし、科学的エビデンスに基づく手法の明示や、体系的なプログラム構成の公開がなく、差別化の余地は大きい。

</div>

### 5.5 ニューロダイバーシティ協会

ニューロダイバーシティの概念に基づく啓発・支援活動。

| 項目 | 内容 |
|------|------|
| 形態 | セミナー、ワークショップ、個別コーチング |
| 対象 | 当事者・企業 |
| 価格 | 初回4,000円、月額20,000〜50,000円 |
| 特徴 | ニューロダイバーシティの視点、強みベース |
| 限界 | ADHDに特化はしていない（ニューロダイバーシティ全般） |

## 8. サービスギャップと差別化機会

調査から特定された4つの主要ギャップ：

### 8.1 ギャップ1: 専門人材の不足

**現状**: ADHDの神経科学的理解 + コーチングスキル + 当事者理解を兼ね備えた日本語コーチがほぼ不在。

- ADDCA/ACO認定の日本語コーチ: ほぼゼロ
- ICF認定 + ADHD専門性: 極少数
- 心理士・カウンセラーはいるが「コーチング」は訓練されていない

<div class="coaching">

**機会**: 日本初のエビデンスベースADHDコーチング認定プログラムの開発。養成したコーチのネットワークがそのままサービス基盤になる。

</div>

### 8.2 ギャップ2: 継続的個別支援の不足

**現状**: 既存サービスは「相談」「就労支援」「療育」が中心で、日常生活全般の**継続的な伴走支援**が欠如。

- 支援センター: 相談は単発〜数回
- Kaien/リタリコ: 就労/療育の目標達成で終了
- 医療機関: 薬物療法の処方が中心。通院頻度は月1回程度

<div class="coaching">

**機会**: 「就職できた後」「診断された後」「薬を飲み始めた後」の**継続的セルフマネジメント支援**。医療でもなく福祉でもない「第三の支援軸」としてのコーチング。

</div>

### 8.3 ギャップ3: グレーゾーン対応の不足

**現状**: 確定診断には至らないが困難を抱える「グレーゾーン」層は、公的支援の対象外。

- 福祉サービス: 手帳/受給者証が必要
- 医療: 確定診断がないと治療対象外
- 推定数: ADHD確定診断者の数倍のグレーゾーン層が存在

<div class="tip">

**差別化のヒント**: コーチングは医療行為ではないため、**診断の有無に関係なく提供可能**。「困っているが制度の網にかからない」層への唯一のアクセスポイントになりうる。

</div>

### 8.4 ギャップ4: 心理的課題への包括的アプローチの不足

**現状**: ADHDの二次障害（低自己肯定感、RSD、完璧主義、慢性的自己否定）に対する統合的な支援がない。

- 医療: 二次的うつ・不安に対する薬物療法
- 心理カウンセリング: ADHDに特化した知見が不十分な場合が多い
- 就労支援: スキル訓練が中心で、心理面は対象外

<div class="coaching">

**機会**: Theme Cで分析した「強みの科学」とTheme Bの「神経科学メカニズム」を統合し、**ADHDのアイデンティティ再構築を含むコーチングプログラム**として提供。「あなたの脳の仕組み」→「だからこの強みがある」→「こう活かせる」の一貫したナラティブ。

</div>

## 9. So What? ポジショニング戦略提言

### 9.1 推奨ポジション

上記の分析から、以下の5要素を組み合わせた**複合ポジショニング**が日本市場の最大の空白地帯である：

```
┌──────────────────────────────────────────────────────┐
│  推奨ポジショニング（5要素の交差点）                      │
```

## 10. 注意点・限界

### 市場データの制約

- 日本のADHDコーチング市場に関する公的統計・市場調査レポートは存在しない
- 各サービスの利用者数・売上データは非公開のものが多い
- 価格情報は公開情報に基づくが、変動の可能性あり（2026年2月時点）

### サービス評価の制約

- ホスピタリッチ等の小規模サービスは公開情報が限定的であり、内容の正確な評価が困難
- デジタルツールの評価はアプリストアのレビュー・公式情報に基づくが、実際の臨床効果のRCTは限定的
- 「効果がある」と「人気がある」は区別する必要がある

### 文化的文脈の重要性

- 日本における発達障害のスティグマ（偏見）は欧米と質が異なる
- 「コーチング」という概念自体の認知度・受容度が市場のサイズを規定する
- 「セルフヘルプ」文化と「専門家に頼る」文化のバランスは国・地域で異なる

<div class="pitfall">

**過大評価のリスク**: 「空白地帯がある」=「市場がある」とは限らない。空白が存在する理由が「ニーズがない」「支払い意思がない」「認知がない」のいずれかを見極める必要がある。ただし、成人ADHD診断数の急増（2020年代の顕著なトレンド）は、潜在ニーズの顕在化を示唆している。

</div>
