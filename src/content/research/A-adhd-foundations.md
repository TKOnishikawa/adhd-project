# A. ADHD の基礎知識

> **調査日**: 2026-02-25
> **調査方法**: Gemini Search Grounding (deep_research + quick_search)
> **目的**: ADHDコーチングメソッド設計のための基礎理解 + 当事者としての自己理解

---

## 1. テーマ概要

ADHDは注意欠如・多動症（Attention-Deficit/Hyperactivity Disorder）の略称であり、不注意、多動性、衝動性を主要な特徴とする神経発達症である。かつては「子どもの障害」とみなされていたが、現在では**成人期にも持続する生涯にわたる特性**であることが国際的コンセンサスとして確立されている（Faraone et al., 2021）。

このテーマが重要である理由は3つある：

1. **コーチングの前提知識**: ADHDの病型によって困りごとのパターンが異なり、アプローチを変える必要がある
2. **過小診断の理解**: 特に女性・成人において見逃されやすい構造を理解しなければ、適切な支援につながらない
3. **当事者の自己理解**: 遅延診断を受けた当事者が自分自身を理解し直すための基盤となる

<div class="tip">

**なぜ「基礎」から始めるのか**: 神経科学（[Theme B](/adhd-project/research/B-neuroscience-mechanism)）や実生活への影響（[Theme F](/adhd-project/research/F-real-life-impact)）を深く理解するためには、まず診断基準・疫学・併存症という「地図」を持っておく必要がある。地図なき旅は遠回りになる。

</div>

## 3. DSM-5-TR 診断基準と3つの病型

### 3.1 診断基準の概要

DSM-5-TR（2022年改訂）における ADHDの診断は、以下の主要な基準から構成される：

**A基準（症状）**:
- **不注意症状**: 9項目のうち、子どもは6つ以上、**成人（17歳以上）は5つ以上**
- **多動性・衝動性症状**: 9項目のうち、子どもは6つ以上、**成人（17歳以上）は5つ以上**
- いずれも**6ヶ月以上**持続していること

**B基準（発症時期）**: 不注意または多動性・衝動性の症状のいくつかが**12歳以前**から存在

**C基準（場面の広がり）**: 2つ以上の場面（家庭、学校/職場、友人関係など）で障害が明確

**D基準（機能障害）**: 社会的、学業的、または職業的機能に臨床的に明確な障害がある

**E基準（除外）**: 統合失調症や他の精神病性障害の経過中に生じたものではなく、他の精神疾患でよりよく説明されない

<div class="pitfall">

**注意: 成人診断の閾値変更**: DSM-5以降、成人（17歳以上）の症状基準は子ども（6/9）より緩和されて**5/9**となった。これは成人期の症状発現が子どもとは異なる形をとることが多いためだが、それでも見逃しが多い。特に不注意優勢型の女性では、「問題行動」として目立ちにくいため、閾値を超えていても受診につながらないケースが多い。

</div>

### 3.2 不注意症状の9項目

| # | 症状 | 成人での典型的な現れ方 |
|---|------|----------------------|
| 1 | 細部への不注意、ケアレスミス | 書類の誤り、メールの誤送信、重要事項の見落とし |
| 2 | 注意の持続困難 | 長時間の会議・読書で集中が途切れる |
| 3 | 直接話しかけられても聞いていない | 相手の話の途中で別のことを考えている |
| 4 | 指示に従えず、完遂できない | プロジェクトの途中で別の仕事に移ってしまう |
| 5 | 課題や活動の整理困難 | デスク周りの混乱、スケジュール管理の苦手さ |
| 6 | 持続的努力を要する課題の回避 | 報告書作成、経費精算などの先延ばし |
| 7 | 必要なものの紛失 | 鍵、財布、スマホ、書類の頻繁な紛失 |
| 8 | 外部刺激による注意散漫 | 通知音、周囲の会話で簡単に注意がそれる |
| 9 | 日常活動での忘れ | 約束の忘却、請求書の支払い忘れ |

### 3.3 多動性・衝動性症状の9項目

| # | 症状 | 成人での典型的な現れ方 |
|---|------|----------------------|
| 1 | 手足をそわそわ動かす | 会議中に貧乏ゆすり、ペン回し |
| 2 | 着席が期待される場面での離席 | 長時間の着席に耐えられずトイレに立つ |
| 3 | 不適切な場面での走り回り・高い所へ登る | 成人では**内的な落ち着きのなさ**として現れる |
| 4 | 静かに遊ぶ・活動することの困難 | 常に何かをしていないと落ち着かない |
| 5 | 「突き動かされるように」活動 | 複数プロジェクトを同時並行、休めない |
| 6 | しゃべりすぎ | 会話の場面で過度に長く話してしまう |
| 7 | 質問が終わる前に答え始める | 相手の文章を先に完成させてしまう |
| 8 | 順番を待つことの困難 | 行列や渋滞へのイライラ、交通違反 |
| 9 | 他者を妨害・侵入 | 会話への割り込み、他人のものを勝手に使う |

### 3.4 3つの病型（発現様式）

DSM-5-TRでは、ADHDは単一の障害の中で3つの**発現様式（presentation）**に分類される。なお、DSM-5以前は「サブタイプ（subtype）」と呼ばれていたが、発現様式は時間経過で変化しうるため、現在は「presentation（発現様式）」という用語が使われている。

```
                    ┌─────────────────────────┐
                    │         ADHD            │
                    │  注意欠如・多動症        │
                    └───────┬─────────────────┘
                            │
            ┌───────────────┼───────────────┐
            ▼               ▼               ▼
   ┌────────────────┐ ┌────────────────┐ ┌────────────────┐
   │  不注意優勢型   │ │    混合型      │ │多動性-衝動性   │
   │  Inattentive   │ │   Combined     │ │優勢型          │
   │                │ │                │ │ Hyperactive-   │
   │ 不注意≧5/9    │ │ 両方≧5/9     │ │ Impulsive      │
   │ 多動-衝動<5/9  │ │                │ │                │
   │                │ │                │ │ 不注意<5/9    │
   │ 女性に多い     │ │ 最も多い       │ │ 多動-衝動≧5/9 │
   │ 見逃されやすい │ │ 診断されやすい │ │ 成人では稀     │
   └────────────────┘ └────────────────┘ └────────────────┘
```

![ADHDの基礎知識マップ](/adhd-project/images/research/A-adhd-foundations.svg)

#### 不注意優勢型（Predominantly Inattentive Presentation）

- **頻度**: 臨床サンプルの約20-30%、コミュニティサンプルではより多い可能性
- **特徴**: 静かな注意困難。ぼんやり、忘れっぽさ、整理の苦手さが主体
- **問題**: 「問題行動」を起こさないため、特に女性では見逃されやすい
- **成人期**: 先延ばし、書類の山、マルチタスク困難として現れる

#### 混合型（Combined Presentation）

- **頻度**: 臨床サンプルで最も多い（約50-70%）
- **特徴**: 不注意と多動性-衝動性の両方が基準を満たす
- **問題**: 症状が多面的で、最も機能障害が大きい傾向
- **成人期**: 落ち着きのなさ + 注意困難 + 衝動的決断の複合

#### 多動性-衝動性優勢型（Predominantly Hyperactive-Impulsive Presentation）

- **頻度**: 最も少ない（約5-10%）。特に成人では稀
- **特徴**: 動き回り、衝動的行動が主体。注意面は比較的保たれる
- **問題**: 幼児期に多く見られるが、多くは成長とともに混合型に移行
- **成人期**: 内的多動（頭の中が忙しい）、衝動買い、リスクテイキング

<div class="coaching">

**コーチングへの示唆: 病型別アプローチ設計**

- **不注意優勢型**: 構造化（チェックリスト、ルーチン）、環境整備（ノイズ遮断）、外部記憶ツール
- **混合型**: 構造化 + エネルギー管理（運動の組み込み）、衝動性への対処スキル
- **多動性-衝動性優勢型**: エネルギーの建設的な方向づけ、意思決定の「一呼吸」テクニック
- **重要**: 病型は時間とともに変化しうるため、定期的な再アセスメントが必要

</div>

## 4. 疫学データ

### 4.1 世界の有病率

| 対象 | 有病率 | 出典 |
|------|--------|------|
| 子どもの世界的有病率 | 5-7% | Polanczyk et al., 2007; Thomas et al., 2015 |
| 成人の世界的有病率 | **2.5-5%** | Faraone et al., 2021 (Consensus Statement) |
| 成人の持続率 | 約50-65% | 子どもの約半数以上が成人期にも基準を満たす |
| 閾下持続率 | さらに多い | 完全基準を満たさなくとも機能障害が続く |

### 4.2 日本の有病率

| 指標 | 数値 | 出典 |
|------|------|------|
| 成人の推定有病率 | **1.65%** | Someya et al., 2015 |
| スクリーニング陽性率（ASRS） | **5.1%** | 一般成人集団 |
| 子どもの推定有病率 | 7-10% | 教師/親評価による |
| 精神科受診者中の有病率 | 高いが正確な統計なし | - |

<div class="tip">

**日本の有病率が低い理由の考察**: 日本の推定有病率1.65%は世界平均（2.5-5%）を大幅に下回るが、スクリーニング陽性率5.1%との乖離が大きい。これは以下の要因が考えられる：
- 診断基準の文化的適合性の問題（多動性の表現が文化によって異なる）
- 受診率の低さ（成人ADHDの認知度がまだ低い）
- 「我慢」「努力不足」として処理される文化的バイアス
- スクリーニング陽性と臨床診断のギャップ

</div>

### 4.3 遺伝と環境

ADHDは高い遺伝性を持つ神経発達症である：

- **遺伝率**: **76%**（双生児研究のメタ分析、Faraone et al., 2005; 2021で再確認）
- **GWAS研究**: 12のリスク遺伝子座が同定（Demontis et al., 2019）、個々の効果量は小さい
- **多因子遺伝**: 数百から数千の遺伝子変異が複合的に寄与する**ポリジェニック**な性質
- **環境要因**: 早産・低出生体重、妊娠中の喫煙・飲酒、鉛曝露、重度の早期剥奪（neglect）

<div class="pitfall">

**注意: 「遺伝率76%」の正しい理解**: 76%とは「ADHDの人の76%が遺伝で決まる」という意味ではない。**集団レベルで、ADHDの表現型のばらつきのうち76%が遺伝的要因で説明できる**という統計的指標である。個人レベルの予測には使えない。また、エピジェネティクスの影響は遺伝率の推定に含まれていない可能性がある。

</div>

## 5. 併存症マッピング

ADHDは単独で存在することのほうが**例外的**であり、多くの場合、他の精神疾患を併存する。これがADHDの診断を難しくし、治療を複雑にする最大の要因のひとつである。

### 5.1 主要な併存症と頻度

| 併存症 | 成人ADHDでの頻度 | ADHDとの関係 |
|--------|-----------------|-------------|
| **不安症（GAD等）** | **約50%** | ADHD起因のストレス蓄積 → 慢性不安 |
| **うつ病** | **20-40%** | 慢性的失敗体験 → 二次性うつ、または共通の神経生物学的基盤 |
| **ASD（自閉スペクトラム症）** | 推定20-50% | DSM-5から併存診断が可能に。実行機能障害が重複 |
| **SLD（限局性学習症）** | 25-40% | 読字障害、書字障害が注意困難と混同されやすい |
| **物質使用障害** | 15-25% | 衝動性 + セルフメディケーション |
| **睡眠障害** | 50-75% | 概日リズムの偏り、入眠困難、過覚醒 |
| **気分調節障害/DMDD** | 高頻度 | 情動制御の困難はADHDの中核症状の一つ |

### 5.2 併存症の構造

```
                        ┌──────────┐
                        │   ADHD   │
                        └────┬─────┘
                 ┌───────────┼───────────┐
                 │           │           │
        ストレス蓄積    衝動性     実行機能障害
                 │           │           │
                 ▼           ▼           ▼
         ┌──────────┐ ┌──────────┐ ┌──────────┐
         │  不安症   │ │ 物質使用 │ │   ASD    │
         │  うつ病   │ │ リスク行動│ │   SLD    │
         └──────────┘ └──────────┘ └──────────┘
                 │
                 ▼
         ┌──────────────┐
         │ 睡眠障害      │
         │（双方向的影響）│
         └──────────────┘
```

<div class="coaching">

**コーチングへの示唆: 併存症スクリーニングの重要性**

- コーチングの初回アセスメントで、少なくとも不安症・うつ病・睡眠障害のスクリーニングを行う
- 併存症が未治療の場合、コーチングの効果が著しく低下する
- **コーチの役割の境界**: 併存症の「診断」はコーチの職域外。気づきを共有し、医療専門家への橋渡しを行う
- ASD併存の場合、感覚過敏や社会的コミュニケーションの困難にも配慮した環境設計が必要
- 睡眠改善はADHD症状の軽減に直結するため、最優先で取り組むべき領域のひとつ

</div>

<div class="pitfall">

**注意: 診断の重複と見分けの難しさ**: 不安症やうつ病の症状（集中困難、エネルギー低下、忘れっぽさ）はADHDの症状と見分けがつきにくい。そのため、ADHDが不安症やうつ病と**誤診**されるケースも多い。特に成人女性では、うつ病の治療を受け続けても改善しない場合、背景にADHDが隠れている可能性を考慮すべきである。

</div>

## 6. 性差と女性の過小診断

### 6.1 性差の数値

| 指標 | 数値 | 含意 |
|------|------|------|
| 子どもの診断比率（男女） | **2-3 : 1** | 男児のほうが多く診断される |
| 成人の診断比率（男女） | **1.5 : 1** | 差が縮小する → 女性の見逃しが成人で追いつく |
| 不注意優勢型の比率（女性） | 男性より高い | 「おとなしいADHD」として見逃される |
| 女性の平均診断年齢 | 男性より**数年遅い** | 学業・キャリアへのダメージが蓄積 |

### 6.2 なぜ女性は見逃されるのか

女性のADHDが過小診断される構造的要因は複合的である：

1. **症状の現れ方の違い**: 女性は不注意優勢型が多く、教室や職場で「問題行動」を起こしにくい
2. **社会的期待と順応圧力**: 女性には「整理整頓」「気配り」「空気を読む」ことが強く期待され、これらの困難は「性格の問題」として処理される
3. **マスキング能力の高さ**: 社会的スキルの学習が早く、困難を隠す技術を早期に身につける（後述）
4. **併存症への注目**: 二次的に発症した不安症やうつ病が治療の焦点となり、ADHDが見過ごされる
5. **診断基準のバイアス**: 初期のADHD研究は男児を対象としたものが圧倒的に多く、基準が男性表現型に偏っている
6. **ホルモンの影響**: エストロゲンの変動（月経周期、妊娠、閉経）がADHD症状を増悪させるが、研究が限られている

### 6.3 女性のADHDに特徴的なパターン

| 領域 | 女性に多い表現 | 男性に多い表現 |
|------|---------------|---------------|
| 注意 | ぼんやり、白昼夢、忘れ物 | 注意散漫、課題の切り替え困難 |
| 多動 | 内的多動（頭の中が忙しい）、おしゃべり | 外的多動（動き回る）、落ち着きのなさ |
| 衝動 | 感情的爆発、衝動買い | 身体的行動の衝動性、リスクテイキング |
| 対処 | 過剰適応、マスキング、完璧主義 | 行動化、反抗、攻撃性 |
| 二次影響 | 内在化（不安、うつ、自責） | 外在化（行為障害、反抗挑発） |

<div class="coaching">

**コーチングへの示唆: 女性クライアントへのアプローチ**

- 「あなたの困難はADHDの特性であって、あなたの努力不足ではない」という明確なメッセージが特に重要
- 過剰適応・マスキングの「成功」をまず認めつつ、その長期的コストを一緒に振り返る
- 月経周期とADHD症状の関係を追跡する提案（症状日記）
- 完璧主義からの解放 → 「good enough（十分に良い）」基準の導入
- 社会的支援ネットワーク（同性のADHDピアグループ）の紹介

</div>

## 7. マスキングのメカニズムと影響

### 7.1 マスキング（Masking / Camouflaging）とは

マスキングとは、ADHDの症状や困難を意図的・無意識的に隠蔽するために、代償的な戦略を過剰に用いることを指す。ASD研究で広く使われてきた「カモフラージュ」概念がADHDにも適用されるようになった。

### 7.2 マスキングの具体的戦略

| カテゴリ | 戦略 | 内側で起きていること |
|---------|------|---------------------|
| **過剰な補償行動** | チェックリストの多重化、二重三重の確認 | 「ミスしたら終わり」という恐怖 |
| **過剰努力** | 人の3倍の時間をかけて同じ成果を出す | 慢性的疲労、睡眠時間の犠牲 |
| **社会的模倣** | 他者の行動パターンをコピー | 自分が何者かわからなくなる |
| **感情の抑制** | 感情的反応を飲み込む、表情管理 | 内面の嵐、帰宅後の崩壊 |
| **環境の徹底管理** | すべてに定位置、ルーチンの固定 | 予想外の事態でパニック |
| **自制と先読み** | 衝動を徹底的に押さえ込む | 意思決定の極端な遅延 |
| **徹夜** | 締め切り前の集中作業 | 慢性的睡眠負債 |

### 7.3 マスキングの長期的影響

```
マスキングの悪循環

  「問題なく見える」 ──→ 周囲が困難を認識しない
        │                         │
        ▼                         ▼
  支援が得られない ◄──── 「大丈夫でしょ？」
        │
        ▼
  さらにマスキングを強化
        │
        ▼
  ┌─────────────────────────┐
  │ 長期的な帰結:            │
  │ - 慢性的疲労・燃え尽き   │
  │ - アイデンティティ喪失   │
  │ - うつ病・不安症の悪化   │
  │ - 自己効力感の低下       │
  │ - 人間関係の表面化       │
  │ - 身体症状（頭痛等）     │
  └─────────────────────────┘
```

### 7.4 マスキングと診断の関係

マスキングが高度に発達した人ほど、臨床場面でも「問題がない」ように見えるため：

- 専門家すら見逃すことがある
- 機能レベルが「高い」と評価され、診断基準Dの「機能障害」を満たさないと判断される
- 本人も「自分はADHDではない」と思い込み、受診しない
- 結果として、**限界に達して崩壊（バーンアウト）するまで診断されない**

<div class="coaching">

**コーチングへの示唆: マスキング解除支援**

マスキングの「解除」は慎重に行う必要がある。長年の防衛戦略を一気に手放すことは危険：

1. **安全な場の構築**: まず「マスクを外しても大丈夫」という信頼関係
2. **マスキングの棚卸し**: どんな戦略を使っているか、それぞれのコストはどのくらいか
3. **段階的な調整**: 「すべて外す」ではなく「コストの高いものから代替策に置き換える」
4. **環境調整の優先**: マスキングが必要ない環境を作ることが最優先（合理的配慮の申請等）
5. **アイデンティティワーク**: 「マスクを外した自分は誰か？」という問いに寄り添う

</div>

## 8. 遅延診断のインパクト

### 8.1 遅延診断（Late Diagnosis）とは

成人期（特に20代後半以降）になって初めてADHDの診断を受けること。特に不注意優勢型の女性に多い。「遅延」とは、症状は幼少期から存在していたが、診断に至らなかったことを意味する。

### 8.2 遅延診断がもたらす累積的影響

遅延診断の影響は「失った時間」だけでなく、自己認識・人間関係・キャリアの全領域に及ぶ：

| 領域 | 累積的影響 | メカニズム |
|------|-----------|-----------|
| **学業** | 学歴が能力に見合わない、中退 | 注意困難 + 適切な支援の欠如 |
| **キャリア** | 転職の繰り返し、能力を発揮できない | 職場適応の困難 + マスキングの限界 |
| **精神的健康** | 慢性的不安、うつ、自己効力感の低下 | 「なぜ自分だけできないのか」の蓄積 |
| **人間関係** | 親密な関係の維持困難、孤立 | 約束の忘却、感情的反応、衝動的発言 |
| **自己認識** | 「怠け者」「能力がない」という自己概念 | 子どもの頃からの否定的フィードバックの内在化 |
| **経済** | 衝動買い、計画的貯蓄の困難 | 衝動性 + 実行機能障害 |

### 8.3 診断後の心理的プロセス

遅延診断を受けた成人は、しばしば以下の心理的プロセスを経る：

```
1. 安堵（Relief）
   「自分のせいじゃなかった」「名前がついた」
       │
       ▼
2. 怒り・悲嘆（Anger/Grief）
   「もっと早く診断されていれば」「失った時間」
       │
       ▼
3. アイデンティティの再構築（Identity Reconstruction）
   「ADHDの自分」をどう統合するか
       │
       ▼
4. 受容と前進（Acceptance & Moving Forward）
   特性を理解した上での新しい生き方の構築
```

<div class="tip">

**当事者の声に多い表現**: 「人生のマニュアルがようやく手に入った」「ずっと水中で呼吸しようとしていたことに気づいた」「パズルのピースがはまった」 -- これらの比喩は、遅延診断がもたらす「安堵」のフェーズを象徴している。

</div>

### 8.4 遅延診断後の特有の課題

- **過去の再解釈**: すべての過去の困難をADHDのレンズで見直す「過剰帰属」のリスク
- **周囲の無理解**: 「今まで大丈夫だったのに」「言い訳では？」という反応
- **治療への期待と現実のギャップ**: 診断=即座の改善ではない
- **マスキングの習慣**: 診断後もマスキングの習慣が抜けない
- **「高機能」ラベル**: 「これまで機能していたのだから大丈夫」という誤解

<div class="coaching">

**コーチングへの示唆: 遅延診断後のアイデンティティ再構築支援**

1. **悲嘆のプロセスを尊重**: 「もっと早く」という怒りや悲しみを否定せず、十分に表現する場を提供
2. **ナラティブ再構築**: 「欠陥のある自分」から「異なる神経回路を持つ自分」への物語の書き換え
3. **強みの再発見**: ADHDの特性が**活かされていた場面**を一緒に特定する
4. **実用的な第一歩**: 感情処理と並行して、日常生活を改善する小さな構造化を始める
5. **ピアサポート**: 同じ経験を持つ当事者コミュニティへの接続
6. **家族への心理教育**: パートナーや家族がADHDを理解するための情報提供

</div>

## 9. So What? コーチングへの統合的示唆

### 9.1 基礎知識がコーチングに与える4つの設計原則

本テーマの知見から、ADHDコーチングの設計において以下の原則が導かれる：

#### 原則1: 病型別アプローチ設計

クライアントの発現様式を初回アセスメントで把握し、介入の重点を調整する：

| 病型 | 主な困りごと | コーチングの重点 |
|------|-------------|-----------------|
| 不注意優勢 | 忘れ物、先延ばし、整理困難 | 外部構造化、ルーチン設計 |
| 混合型 | 上記 + 落ち着きのなさ、衝動性 | 構造化 + エネルギー管理 + 衝動性対処 |
| 多動-衝動優勢 | 衝動的決断、リスクテイキング | 意思決定プロセスの構造化 |

#### 原則2: 併存症スクリーニング

- 初回セッションで不安・うつ・睡眠の簡易スクリーニングを実施
- 併存症の存在を前提とした現実的な目標設定
- 必要に応じて医療専門家との連携体制を構築

#### 原則3: マスキング解除支援

- 「安全な場」としてのコーチングセッションの位置づけ
- マスキングの棚卸しとコスト分析
- 段階的な代替策の導入

#### 原則4: アイデンティティ再構築支援（遅延診断者向け）

- 悲嘆プロセスの尊重と伴走
- 強みベースのナラティブ再構築
- 実用的スキルと心理的プロセスの並行支援

### 9.2 神経科学との接続

これらの基礎知識は、[Theme B: 神経科学メカニズム](/adhd-project/research/B-neuroscience-mechanism)で扱う実行機能障害、ドーパミン仮説、DMN介入仮説と密接に結びつく。例えば：

- 不注意優勢型の「ぼんやり」→ DMNの過剰活動（Theme B: DMN介入仮説）
- 衝動性 → 行動抑制の障害（Theme B: バークレーモデル）
- 併存症の発症メカニズム → 報酬系の機能不全（Theme B: ドーパミン仮説）

### 9.3 実生活への影響との接続

基礎知識で整理した病型・併存症・マスキングは、[Theme F: 実生活への影響](/adhd-project/research/F-real-life-impact)で扱う職場適応、金銭管理、人間関係の具体的な困難と直結する。

## 12. 注意点・限界

- **DSM-5-TRは万能ではない**: 診断基準は臨床ツールであり、人間の複雑さを完全に捕捉するものではない。特に成人・女性・高IQの個人では感度が低い
- **有病率データの不確実性**: 調査方法（自己報告 vs 臨床面接）、文化、基準のバージョンにより大きく変動する
- **日本固有のデータの不足**: 日本の成人ADHDに関する大規模疫学調査は限られている
- **併存症の因果関係**: ADHDと併存症の因果関係は完全には解明されておらず、共通の遺伝的・神経生物学的基盤を持つ可能性がある
- **マスキング研究の初期段階**: ADHD文脈でのマスキング研究はASD研究に比べてまだ少なく、エビデンスレベルは中程度
- **遅延診断の影響に関するエビデンス**: 主に後方視的研究や質的研究に基づいており、前方視的縦断研究は少ない
- **文化的バイアス**: 診断基準・研究の多くが欧米中心であり、日本を含むアジア文化圏への適用には注意が必要

## 13. 未解決の問い

- **閾下ADHD（Subthreshold ADHD）**: 完全な基準を満たさないが機能障害がある人々へのアプローチは？
- **ホルモンとADHD**: エストロゲン・プロゲステロンの変動がADHD症状に与える影響の詳細なメカニズム
- **高齢者のADHD**: 加齢に伴うADHD症状の変化と認知症との鑑別
- **文化横断的な診断基準**: 東アジア文化圏に適合したADHD診断基準・スクリーニングツールの開発
- **マスキングの定量化**: マスキングの程度を測定するADHD固有の尺度の開発
- **診断タイミングの最適化**: 早期診断と適切な支援がもたらすアウトカム改善の定量的エビデンス
- **デジタルフェノタイピング**: スマートフォンやウェアラブルデバイスによるADHD症状の客観的測定は実用化できるか
