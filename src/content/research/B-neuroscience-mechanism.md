# B. ADHD 神経科学メカニズム

> **調査日**: 2026-02-25
> **調査方法**: Gemini Search Grounding (deep_research + quick_search)
> **目的**: ADHDコーチングメソッド設計のための神経科学的基盤理解 + 当事者としての自己理解

---

## 1. テーマ概要

ADHDは「意志の弱さ」や「性格の問題」ではなく、脳の自己調整システムにおける明確な神経生物学的特性に基づく。主に以下の領域で機能的な違いが認められている：

- **前頭前野（PFC）**: 実行機能の中枢。計画、抑制、ワーキングメモリを司る
- **基底核（線条体）**: 報酬処理、動機づけ、運動制御
- **小脳**: 時間処理、運動制御
- **神経伝達物質**: ドーパミン、ノルエピネフリンの調節異常
- **脳内ネットワーク**: デフォルトモードネットワーク（DMN）とタスクポジティブネットワーク（TPN）の切り替え不全

---

## 2. 主要理論・モデル

### 2.1 バークレーの行動抑制モデル（Barkley, 2001）

**核心**: ADHDの中核的欠陥は「行動抑制の障害」であり、これが4つの実行機能の発達不全に連鎖する。

```
行動抑制の障害
  ├── 非言語性ワーキングメモリ（映像記憶・未来予測）
  ├── 言語性ワーキングメモリ（内言・セルフトーク）
  ├── 情動・動機づけ・覚醒の自己調整
  └── 再構成（分析・統合・新しい行動の創造）
```

**コーチングへの示唆**:
- 「なぜやらないか」ではなく「どうすれば実行できるか」に焦点
- **情報の外部化**（リマインダー、タイマー、チェックリスト）が不可欠
- コーチとの対話で思考を言語化 → 内言の弱さを補完

**批判・限界**:
- ADHDの人すべてが実行機能障害を持つわけではない
- ADHDでない人にも実行機能障害がある場合がある
- ADHDの唯一の原因モデルではなく、すべての症状を説明するものではない
- 前頭葉と皮質下異常との関連性主張が時期尚早との批判も

### 2.2 ブラウンの6クラスターモデル（Brown, 2005）

**核心**: ADHDを単一の障害ではなく、6つの相互関連する認知管理クラスターの機能不全として捉える。

| クラスター | 内容 | 日常への影響 |
|-----------|------|-------------|
| 活性化 | タスク開始、優先順位 | 先延ばし、取り掛かれない |
| 焦点化 | 注意維持、切り替え | 注意散漫、過集中 |
| 努力 | 覚醒維持、処理速度 | 退屈で眠くなる |
| 情動 | 欲求不満管理 | 感情の爆発、RSD |
| 記憶 | ワーキングメモリ | 聞き忘れ、忘れ物 |
| 行動 | 行動監視・自己調整 | 衝動的行動 |

**コーチングへの示唆**:
- クライアントの困難を具体的なクラスターに分類 → 的を絞った介入設計
- 「活性化」→ ベイビーステップ（タスクの極限分割）
- 「焦点化」→ 環境整備、ノイズキャンセリング
- 「努力」→ ポモドーロ・テクニック
- 「情動」→ 感情の波の自己認識 + 事前対処法準備

**信頼性メモ**: 広く知られているが、バークレーほど学術論文で直接引用されるケースは少ない（中信頼）

### 2.3 ソヌガ＝バークの二重/三重経路モデル（Sonuga-Barke, 2002/2003/2010）

**核心**: ADHDの異質性を複数の神経心理学的経路で説明する。

| 経路 | メカニズム | 脳領域 | 症状の現れ方 |
|------|----------|--------|-------------|
| 実行機能障害経路 (Cool) | 抑制制御・計画の弱さ | 前頭-背側線条体 | 計画性のなさ、WM低下 |
| 遅延回避経路 (Hot) | 待つことへの嫌悪 | 前頭-腹側線条体（報酬系） | 衝動性、即時報酬選好 |
| 時間処理障害経路 | 時間知覚の歪み | 小脳関連回路 | Time Blindness |

**コーチングへの示唆**:
- **アセスメントの鍵**: クライアントがどの経路の問題をより強く抱えているか
- 実行機能障害型 → 構造化・計画支援が中心
- 遅延回避型 → ゲーム化、頻繁な即時フィードバック、動機づけ焦点

## 3. 神経化学（Neurochemistry）

### 3.1 ドーパミン仮説

ADHDの脳ではドーパミン伝達システムに異常がある。特に報酬予測・動機づけ・学習に関与するドーパミンの働きがアンバランス。

**トニック vs フェイジック ドーパミン**:
| 種類 | 説明 | ADHDでの異常 |
|------|------|-------------|
| トニック（持続的） | シナプス間の低レベル持続放出。覚醒・S/N比 | 低い → 退屈なタスクで集中困難 |
| フェイジック（一時的） | 予測外の報酬で大量放出。学習・強化 | DAT密度高 → 再取り込みが早すぎ → 報酬信号が弱い |

**コーチングへの示唆**:
- 「やる気がない」のではなく「脳が報酬を感じにくい」状態
- タスクのゲーム化、目新しさの導入、競争要素
- 大きな目標を細分化 → 小さな達成ごとに報酬設定

**反証・批判**:
- 「ドーパミン欠乏」は単純化しすぎ。脳のすべての領域で減少しているわけではない
- 一部の領域ではドーパミン過剰の可能性も
- ドーパミンレベルそのものより、受容体のバリエーションによる**信号の強さの問題**が大きい可能性
- 「ドーパミン作動性低下状態そのもの」がADHD主要構成要素である証拠は限定的（Madsen et al., 2024）

### 3.2 ノルエピネフリンの役割

- 覚醒、注意、警戒心を制御
- 前頭前野で無関係刺激をフィルタリング（S/N比改善）
- ADHDではこのシステムも機能不全 → 注意散漫、覚醒不安定

**コーチングへの示唆**:
- 環境整備の重要性（物理的 + デジタル）
- 適度な運動がノルエピネフリン調整に有効

## 4. 報酬系（Reward System）

### 4.1 遅延割引（Delay Discounting）

報酬を得るまでの時間が長くなるほど、主観的価値が急激に割り引かれる。ADHDでは割引率が定型発達より急。

```
定型発達:  「今の1万円」≒「3ヶ月後の1.2万円」（緩やかな割引）
ADHD:      「今の1万円」>> 「3ヶ月後の5万円」（急激な割引）
```

**コーチングへの示唆**:
- **報酬の即時化**: 長期目標の達成過程に即時的なご褒美
- **進捗の可視化**: グラフ、シール、プログレスバー
- **ネガティブ結果の具体化**: 先延ばしの不利益を具体的にイメージさせる

### 4.2 興味に基づく神経系（Interest-Based Nervous System）

ウィリアム・ドッドソン提唱。ADHDの脳は「重要性」ではなく「興味」で駆動される。

**活性化の4トリガー**:
1. **興味（Interest）**: 心から面白いと思える
2. **目新しさ（Novelty）**: 新しい刺激
3. **挑戦（Challenge）**: 適度な難易度
4. **緊急性（Urgency）**: 締め切り直前

**コーチングへの示唆**:
- 意志力で退屈なタスクに立ち向かわせるのではなく、4トリガーのいずれかに結びつける
- タスクと興味の連結、ゲーム化、人為的な緊急性の創出（ボディダブル）

**科学的検証状況**:
- 大規模な査読付き研究による直接検証は見つかっていない
- 臨床観察に基づくモデルだが、ドーパミン報酬系の知見と整合性が高い
- 「影響力のある臨床モデル」として広く支持（信頼性: 中〜高）

## 5. 時間処理（Temporal Processing）

### 時間盲（Time Blindness）

- 前頭前野、基底核、小脳の連携による時間処理が機能不全
- ドーパミン作動性回路 + ワーキングメモリの弱さが関与
- 時間再生産課題・推定課題での成績が不正確でばらつき大

**コーチングへの示唆**:
- **アナログ時計**: 時間経過が視覚的にわかる
- **タイムタイマー**: 残り時間が視覚的に減少
- **未来の現在化**: 「あと〇回の週末」のような体感単位
- **バッファータイム**: 移動時間・作業時間に意図的余裕

---

## 6. デフォルトモードネットワーク（DMN）

### DMN介入仮説

- **DMN**: 安静時に活発化する脳内ネットワーク（マインドワンダリング、自己参照）
- **TPN**: 課題遂行時に活発化するネットワーク
- 通常はDMN↔TPNがシーソーのように切り替わる（反相関）
- **ADHDでは**: 課題中にDMNが過剰に活動し続ける → 集中時に無関係な考えが浮かぶ

**Norman et al. (2022) メガ解析 (N>2600)**:
- ADHD診断とDMN-TPN間の反相関低下が関連
- 効果量は小さいが、連続的特性として脳ネットワークレベルで確認
- DMN介入仮説を強力に支持

**反証チェック結果**:
- DMN介入仮説に反する有意な証拠は現時点で見つかっていない
- むしろ支持する証拠が蓄積されている（高信頼）

**コーチングへの示唆**:
- **マインドフルネス**: 注意の所在を客観的に認識 → 意図的に課題に引き戻す訓練
- **外部メモ**: 頭の中で情報保持不要に → DMN活性化余地を減少
- **興味の活用**: 興味ある課題ではDMN介入が起こりにくい

---

## 7. ワーキングメモリ（Working Memory）

### ADHDにおける特異的障害

「記憶力が悪い」ではなく、**情報を保持しながら干渉を遮断し目標関連操作を行うこと**が困難。

**具体的影響**:
- 複数ステップの指示を保持できない
- 長期目標のステップ順序を思い描けない
- 「持ち物を確認する」を頭に置いておけない
- 結果予測前に行動してしまう（衝動性）

**他の実行機能との関係**:
- WMは他のすべての実行機能の**基盤**
- 計画・整理には、まずWM内に情報を保持・操作する必要
- WM障害 → 実行機能全般に波及

**コーチングへの示唆**:
- WMは容量の小さい「脳の作業台」→ 過負荷を避ける
- **外部化ツール徹底**: メモ帳、リマインダー、チェックリスト、ホワイトボード
- **指示の単純化**: 一度にひとつ、簡潔に
- **ルーチン化・自動化**: 頻繁な作業を定型化 → WM不使用で遂行
- **環境の単純化**: 視覚的ノイズ軽減（散らかった部屋 → WM負荷）

## 5. コーチングメソッド設計への示唆（統合）

### 設計原則（神経科学に基づく）

1. **外部化原則**: WM・内言の弱さ → 情報・計画・感情を外部ツールに出す
2. **報酬即時化原則**: 遅延割引の急さ → 小さな報酬を頻繁に、進捗を可視化
3. **興味駆動原則**: Interest-Based NS → 4トリガー（興味・新奇・挑戦・緊急）に接続
4. **環境設計原則**: NE系・DMN干渉 → 物理/デジタル環境の最適化
5. **時間構造化原則**: Time Blindness → アナログ時計、タイムタイマー、バッファ
6. **経路別個別化原則**: 三重経路モデル → クライアントの主要経路をアセスメント
7. **マインドフルネス原則**: DMN制御 → 注意の自己モニタリング訓練

### クライアントアセスメントの枠組み

```
Phase 1: どの経路が主要か？
  ├── 実行機能障害型 → 構造化・計画支援
  ├── 遅延回避型 → 動機づけ・ゲーム化
  └── 時間処理障害型 → 時間の外部化

Phase 2: どのクラスターが最も困難か？（Brown 6クラスター）
  → 的を絞った介入設計

Phase 3: 興味トリガーの特定
  → クライアント固有の「ドーパミンスイッチ」を発見
```

## 注意点・限界

- いずれのモデルもADHDの全てを説明するものではなく、単純化された理論的枠組み
- 脳画像研究は集団平均の差であり、個人レベルの診断ツールとして確立されていない
- 効果量が小さい研究も多く、ADHD群と定型群の差は重なりの大きいスペクトラム
- ドーパミン仮説は「欠乏」という単純な話ではなく、信号の質の問題
- Interest-Based Nervous Systemは臨床モデルであり、大規模な学術検証は未完了

## 未解決の問い

- コーチングや心理療法が脳機能・ネットワークにどんな変化をもたらすかの縦断的研究が不足
- 成人期、特に女性や高齢者のADHDにおける神経生物学的特徴の解明
- 併存症（不安・うつ）が各メカニズムにどう影響するかの解明
- Dodsonの Interest-Based NS の学術的検証
